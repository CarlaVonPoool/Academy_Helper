# Beispiel Dokument

Dies ist ein Beispiel-Dokument für den RAG Chat Assistant.

## Funktionen

Der RAG Chat Assistant kann:

- Dokumente aus Markdown-Dateien laden
- Intelligente Suche in den Dokumenten durchführen
- Fragen zu den Inhalten beantworten
- Mehrere Dokumente gleichzeitig durchsuchen

## Installation

1. Repository klonen
2. Dependencies installieren: `pip install -r requirements.txt`
3. Streamlit App starten: `streamlit run app.py`

## Nutzung

1. Dokumente in einen Ordner legen
2. Pfad in der Sidebar eingeben
3. "Dokumente laden und indexieren" klicken
4. Fragen im Chat stellen

## Technologie

- **Streamlit**: Web-Interface
- **Sentence Transformers**: Text-Embeddings
- **scikit-learn**: Similarity Search
- **Anthropic Claude**: Antwort-Generierung
