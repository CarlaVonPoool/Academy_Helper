# ⚙️ Mitarbeiter:innen einladen

Neue Mitarbeiter:innen können Sie in Poool direkt über die Benutzerverwaltung einladen. So stellen Sie sicher, dass alle Personen die passenden Zugriffsrechte und Rollen erhalten.

## ✅ Systemvoraussetzungen & Einstellungen

- Sie benötigen **Administratorrechte**.
- Folgende Pflichtfelder müssen ausgefüllt sein:
    - **E-Mail-Adresse** (muss freie Mail Adresse sein)
    - **Vorname**
    - **Nachname**
    - **Rolle**
    - **Funktion**

💡 Wenn Sie Microsoft Single Sign On (SSO) nutzen möchten, wenden Sie sich bitte an [support@poool.cc](mailto:support@poool.cc). Die Einrichtung wird von unserem Team vorgenommen und gilt immer für **alle Benutzer:innen einer Instanz**.

## 📌 Prozessbeschreibung: Schritt-für-Schritt Anleitung

**Schritt 1: Einladung erstellen**

- Gehen Sie zu `Benutzer → Einladung` und tragen Sie die Pflichtfelder ein.
    
    ![image.png](Einladung%20der%20Mitarbeiter%20innen/image.png)
    

**Schritt 2: Mehrere Benutzer:innen hinzufügen**

- Mit `+ Benutzer` können Sie zusätzliche Reihen öffnen.
- Mit `+ mehrere hinzufügen` lassen sich mehrere E-Mail-Adressen auf einmal einfügen (z. B. aus einer **Excel-Liste**).
    
    <aside>
    💡 Die Mail-Adressen müssen entweder durch Absatz, Beistrich oder Stichpunkt getrennt sein.
    
    </aside>
    
    ![image.png](Einladung%20der%20Mitarbeiter%20innen/image%201.png)
    

**Schritt 3: Einladung versenden**

Mit `→ Einladung versenden` wird eine E-Mail verschickt. Die eingeladenen Personen können sich mit **Passwort** oder über **Microsoft SSO** anmelden.

![image.png](Einladung%20der%20Mitarbeiter%20innen/image%202.png)

➡️ Weitere Infos: [Microsoft Single Sign On einrichten](../../Unternehmen%20&%20Mitarbeiter/Mitarbeiteranlage%20&%20Berechtigungen/Microsoft%20Single%20Sign%20on%20einrichten%20d63aa7ff12064ba68e3c4da8625bbc49.md)

**Schritt 4: Benutzerrollen & Zugriffsrechte festlegen**

- Öffnen Sie das Benutzerprofil und gehen Sie zu `Zugriffsrechte`.

![image.png](Einladung%20der%20Mitarbeiter%20innen/image%203.png)

- Wählen Sie eine **vordefinierte Rolle** oder vergeben Sie individuelle Berechtigungen.
- Standardmäßig vorhandene Rollen:
    - Benutzer
    - Projektmanager
    - Zeiterfasser
    - Freelancer
        
        ➡️ Weitere Rollen können unter `Benutzer → Rollen` angelegt oder bearbeitet werden.
        

![image.png](Einladung%20der%20Mitarbeiter%20innen/image%204.png)

## 💼 Anwendungsfälle

- Schnelle Anlage mehrerer Mitarbeiter:innen (z. B. bei Projektstart oder Neueinstellungen).
- Zentrale Verwaltung von Benutzerrollen und individuellen Rechten.
- Nutzung von **Microsoft SSO** für einheitlichen Login.

## ✅ Vorteile im Überblick

- **Effizienz**: Mehrere Benutzer:innen gleichzeitig einladen.
- **Flexibilität**: Rollen & Berechtigungen individuell anpassen.
- **Integration**: Optionale Anbindung an Microsoft Single Sign On.

## 🔍 Zusammenfassung des Prozesses

Über `Benutzer → Einladung` können Sie neue Mitarbeiter:innen schnell und einfach hinzufügen. Sie geben die Pflichtfelder ein, versenden die Einladungen und weisen anschließend Rollen sowie Zugriffsrechte zu. Optional steht die Anmeldung per Microsoft SSO zur Verfügung.
