# ⚙️ Dashboard einrichten

Das **Dashboard** ist Ihre persönliche Startseite in Poool. Hier behalten Sie alle wichtigen Informationen im Blick – von Projekten über Stunden bis zu Rechnungen. In diesem Guide erfahren Sie, wie Sie das Dashboard als Admin für Ihr Team vorbereiten und wie Sie es als Nutzer:in individuell anpassen.

---

## ✅ Systemvoraussetzungen & Einstellungen

- Es sind **keine zusätzlichen Systemeinstellungen** durch den Support erforderlich.
- Zum **Anpassen für alle Benutzer:innen** benötigen Sie **Admin-Rechte**.
- Jede:r Nutzer:in kann das eigene Dashboard **individuell** gestalten, ohne dass dies Auswirkungen auf andere hat.

---

## 📌 Prozessbeschreibung: Schritt-für-Schritt Anleitung

- **Schritt 1: Überblick verschaffen**
    - Das Dashboard besteht aus **Widgets**, die Informationen zu Projekten, Stunden oder Rechnungen anzeigen.
    - Widgets können verschoben, hinzugefügt oder entfernt werden.
- **Schritt 2: Dashboard als Admin konfigurieren**
    - Gehen Sie zu **Setup → Dashboard-Konfiguration**.
    - Wählen Sie die Widgets aus, die **standardmäßig** für alle neuen Nutzer:innen sichtbar sein sollen.
    - Legen Sie fest, ob Widgets für bestimmte **Rollen** (z. B. Projektleiter:in, Buchhaltung) verfügbar sind.
    - Speichern Sie die Konfiguration.
- **Schritt 3: Dashboard als Nutzer:in anpassen**
    - Klicken Sie auf **„Widget hinzufügen“**, um weitere Widgets auszuwählen.
    - Ordnen Sie die Widgets per **Drag & Drop** neu an.
    - Entfernen Sie Widgets über das **X-Symbol**.
    - Ihr persönliches Dashboard ist nur für Sie sichtbar.

---

## 💼 Anwendungsfälle

- **Projektleiter:innen** können sich ihre offenen Projekte und Aufgaben anzeigen lassen.
- **Mitarbeiter:innen** sehen ihre gebuchten Stunden und anstehenden Tätigkeiten.
- **Buchhaltung** kann Widgets für Rechnungen, Zahlungen und offene Posten nutzen.
- Über Links im Dashboard gelangen Sie direkt in die jeweiligen Detailbereiche (z. B. Projekte, Reporting, Abrechnung).

---

## ✅ Vorteile im Überblick

- **Individuelle Übersicht**: Jede:r sieht die für sie relevanten Daten.
- **Rollenbasierte Konfiguration**: Admins können Standard-Setups für verschiedene Rollen definieren.
- **Hohe Flexibilität**: Widgets lassen sich jederzeit hinzufügen, entfernen oder neu anordnen.
