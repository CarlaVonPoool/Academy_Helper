# Aufwandsprojekte

Aufwandsprojekte sind Projekte oder Aufträge, bei denen die Abrechnung auf Basis der tatsächlich erbrachten Leistungen und des dafür aufgewendeten Aufwands erfolgt. Anders als bei Pauschalprojekten, bei denen ein fester Betrag für die gesamte Leistung vereinbart wird, ermöglichen Aufwandsprojekte eine flexible Abrechnung, die den realen Ressourcenverbrauch widerspiegelt.

### Zeiten abrechnen:

Im ersten Schritt wird wie gewohnt ein neues Projekt erstellt. In diesem Projekt wird ein Angebot erstellt und es werden einzelne Leistungen mit einer bestimmten Anzahl an Stunden hinzugefügt. Das Angebot wird gespeichert und beauftragt. Da es sich hier um Aufwandsprojekte handelt, wird bei der Stundenabrechnung “Stunden direkt abrechnen” gewählt.

![Untitled](Aufwandsprojekte/Untitled.png)

Nun ist es an der Zeit, dass an dem Projekt gearbeitet wird und Zeiten auf das entsprechende Projekt gebucht werden.

Sobald es zur Rechnungsstellung kommt, navigieren Sie in das gewünschte Projekt zu dem Raster `Rechnungen` und erstellen Sie mit `+ Ausgangsrechnung` eine neue Ausgangsrechnung. Vergeben Sie einen Titel für die Ausgangsrechnung und navigieren Sie zum `Abrechnungsassistenten`. Mit einem Klick darauf, öffnet sich ein neues Pop-up. Da in diesem Fall die Stunden direkt abgerechnet werden sollen, wählen Sie den Punkt `Stunden und Tickets`. 

![Untitled](Aufwandsprojekte/Untitled%201.png)

Sie sehen dann eine Liste der Stunden die bisher auf dieses Projekt gebucht wurden. Die Stunden die unter dem Punkt “Direkt erfasst” stehen, sind Stunden, die direkt auf das Projekt gebucht wurden. Die anderen Stunden wurden auf dein angelegtes Ticket erfasst. 

Sie können mit dem Kontrollkästchen auf der rechten Seite wählen, ob sie alle gebuchten Stunden verrechnen wollen oder nur einen Teil.

![Untitled](Aufwandsprojekte/Untitled%202.png)

Im nächsten Schritt können Sie wählen, ob Sie die Rechnung einem Auftragsplan zuordnen wollen. Das Häkchen bei `Pauschal` setzen Sie, wenn der Auftrag mit der Abrechnung “geschlossen” werden soll. Wenn der Restwert weiterhin als “offen” stehen bleiben soll, setzen Sie **kein** Häkchen bei `Pauschal`.

![Untitled](Aufwandsprojekte/Untitled%203.png)

Als nächstes wählen Sie, wie die Stunden auf der Rechnung dargestellt werden. In den Abrechnungsvorlagen gibt es dazu mehrere Möglichkeiten:

- **Leistungen**: Die gebuchten Stunden werden als **einzelne Positionen** angeführt.
- **Tickets**: Insofern die Stunden auf ein Ticket gebucht wurden, wird als **Positionstitel** der **Tickettitel** verwendet. Wurden die Stunden direkt auf das Projekt gebucht wird als **Positionstitel** der **Projekttitel** verwendet.
- **Zusammengefasst**: Alle gebuchten Stunden werden auf eine Position **zusammengefasst**.
- **Ticketgruppe mit Leistungen**: Insofern die Stunden auf ein Ticket gebucht wurden, wird der **Tickettitel** als neue **Gruppe** angelegt und die gebuchten **Leistungen** werden als **Position** in der Gruppe angelegt. Wurden die Stunden direkt auf das Projekt gebucht wird als **Gruppentitel** der **Projekttitel** verwendet und die gebuchten **Leistungen** werden als **Positionen** hinzugefügt.
- **Ticketgruppe mit Aufgaben**: Insofern die Stunden auf ein Ticket gebucht wurden, wird der **Tickettitel** als neue **Gruppe** angelegt und die **Aufgaben** werden als **Positionen** hinterlegt. Wurden die Stunden direkt auf das Projekt gebucht wird als **Gruppentitel** der **Projekttitel** verwendet und die gebuchten **Leistungen** werden zu einer **Position zusammengefasst**.

Wenn Sie eine Abrechnungsvorlage wählen, wird Ihnen immer eine Vorschau angezeigt, wie die Stunden auf der Rechnung abgebildet werden. Sie können die Vorlage wählen, die für Ihren Fall am besten passt und jegliche Anpassungen vornehmen wie beispielsweise Anzahl, Einheit oder Preis verändern. Sobald Sie mit der Darstellung zufrieden sind, können Sie die Eingaben mit `→ Weiter` bestätigen.

![Untitled](Aufwandsprojekte/Untitled%204.png)

Sie können im nächsten Schritt die Stunden noch zuteilen. Hier stehen Ihnen entweder die Optionen Eigen- und Fremdleistungen zur Verfügung oder Sie können im Vorhinein Ihre Gruppen anlegen, in die Sie die Stunden einfügen wollen. Mit einem Klick auf das `+` verschieben Sie die Position in die gewünschte Gruppe. Sobald die Stunden am gewünschten Ort sind, können Sie die Stunden mit `→ Zur Rechnung hinzufügen` der Rechnung beifügen.

![Untitled](Aufwandsprojekte/Untitled%205.png)

Sie können dann auch in der Rechnung noch alles Ihren Wünschen entsprechend anpassen und die Rechnung wie gewohnt abrechnen.
