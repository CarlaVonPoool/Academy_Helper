# Freelancer anlegen

Freelancer sind externe Mitarbeitende, die projektbasiert für Ihr Unternehmen arbeiten. In Poool  unterscheiden sich von internen Mitarbeitern bei Rechten, Funktion und Projektzugriff.

Mit dieser Anleitung legen Sie externe Mitarbeitende sauber an, vergeben passende Rollen und steuern exakt, worauf sie zugreifen dürfen.

## **📌 Prozessbeschreibung: Schritt-für-Schritt Anleitung**

### **Schritt 1**: Person im CRM anlegen

- Navigieren Sie zu `CRM → Adressbuch`. Auch externe Mitarbeitende müssen in einer Firma im CRM als Person / Kontakt angelegt werden.
- Wählen Sie die gewünschte Freelancer-Firma aus, oder legen Sie für den neuen Freelancer eine Firma an.
- Klicken Sie auf `+ Person`
- Erfassen Sie Name und E-Mail-Adresse (Pflichtfeld)
- Optional: Telefonnummer, Funktion, Tags oder interne Notizen
    
    ![image.png](Freelancer%20anlegen/image.png)
    

### **Schritt 2**: Benutzer anlegen

- Navigieren Sie zu `Benutzer → Verwaltung` und klicken Sie auf `+ Benutzer`.
- Geben Sie die E-Mail-Adresse des Freelancers ein und legen Sie ein vorläufiges Passwort fest
- Optional aktivieren:
    - *E-Mail-Bestätigung beim ersten Login erforderlich*
    - *Passwortänderung beim ersten Login erzwingen*
- Speichern sie das Profil ab
    
    ![Untitled](Freelancer%20anlegen/Untitled.png)
    

### **Schritt 3: Benutzerrolle und Zugriffsrechte definieren**

- Wählen Sie im Benutzerprofil die Rolle **Freelancer** aus.
    
    ![Untitled](Freelancer%20anlegen/Untitled%201.png)
    

Diese Rolle ist im Standard so ausgelegt, dass externe Mitarbeitende:

- nur projektbezogene Funktionen nutzen
- keine internen Unternehmensdaten einsehen
- Zeit erfassen können (falls aktiviert)
- Legen Sie fest, auf welche Projekte der Freelancer zugreifen darf und welche Tickets (Eigene oder Alle) bearbeitet werden dürfen.

### Schritt 4: Externe Mitarbeiter anlegen

Damit Freelancer korrekt in Zeiterfassung, Projektmanagement und Abrechnung erscheinen, müssen sie als externe Mitarbeiter angelegt werden. 

- Navigieren Sie zu `Unternehmen → Externe Mitarbeiter`
- Klicken Sie auf `+ Externen Mitarbeiter`
- Nehmen Sie die wichtigsten Einstellungen vor:
    - Verknüpfen mit dem oben angelegten Kontakt (Firma und Person)
    - Auswahl einer Funktion

![image.png](Freelancer%20anlegen/image%201.png)

Mit Klick auf Speichern ist der Freelancer angelegt. Sie können ihm nun die Zugangsdaten (E-Mail + vorläufiges Passwort) zum Login zukommen lassen.

---

## 🔍 **Zusammenfassung des Prozesses**

Über `CRM → Adressbuch`, `Benutzer → Verwaltung` und `Unternehmen → Externe Mitarbeiter` legen Sie Freelancer vollständig in Poool an. Von Kontaktdaten über Berechtigungen bis hin zu Projektzugriffen – alle Schritte sind logisch verknüpft und gewährleisten einen sauberen, sicheren Datenbestand für externe Mitarbeitende.
