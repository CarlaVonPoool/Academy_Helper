# Beschreibung Berechtigungen

Article Description: Alle Berechtigungen erklärt
Last Updated: 2. September 2024
Published: Yes
Suggested: No

| Berechtigung | Beschreibung |
| --- | --- |
| Projekte | Zugriff auf die Projektliste inkl. Termine und Status. Inkludiert auch die Möglichkeit, selbst neue Projekte anzulegen. |
| Angebote | Lese- und Schreibrechte auf bestehende Angebotskalkulationen innerhalb der Projekte sowie als Gesamtübersicht.
**Eingeschränkter Zugriff**: Der Benutzer kann nur auf Angebote bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist zugreifen. |
| Aufträge | Lese- und Schreibrechte auf bestehende Auftragskalkulationen innerhalb der Projekte sowie als Gesamtübersicht.
**Eingeschränkter Zugriff**: Der Benutzer kann nur auf Aufträge bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist zugreifen. |
| Rechnungen | Lese- und Schreibrechte auf bestehende Rechnungskalkulationen innerhalb der Projekte sowie als Gesamtübersicht.
**Eingeschränkter Zugriff**: Der Benutzer kann nur auf Rechnungen bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist zugreifen. |
| Rechnungen abrechnen | Berechtigung zur Vergabe neuer Rechnungsnummern. |
| Rechnungen stornieren | Berechtigung um bestehende Rechnungen mit Rechnungsnummer zu stornieren und eine Gutschrift zu erstellen. |
| Kontakte | Lese- oder Lese- und Schreibrecht im Adressbuch - Bearbeitung von Kontakten. |
| Zeiterfassung | Möglichkeit auf Projekte Zeiten zu buchen.
**Nur freigegebene Projekte**: Für Freelancer vorgesehene Sonderberechtigung um einzelne Projekte manuell zur Zeiterfassung freigeben zu können. |
| Unternehmen | Zugriff auf die Mitarbeiterverwaltung. Einsicht in aktuelle Urlaubsstände, Zeitkonten und Bearbeitungsmöglichkeiten. Sensible Informationen (Gehalt, …) sind weiter unten extra geregelt. |
| Benutzer | Zugriff auf den Bereich Benutzer inkl. Rollen und Rechtevergabe für einzelne Personen(-gruppen). |
| System & Einstellungen | Die Bereiche System & Einstellungen regeln grundsätzliche Einstellungen in Poool (Konfigurationen). |
| Tickets | Möglichkeit zur Anlage und Einsicht von Tickets. Das Recht zur Bearbeitung von bestehenden Tickets wird über die Einstellungen zu Tickets auf Ersteller, Verantwortliche oder Alle geregelt.
**Nur freigegebene Projekte**: Für Freelancer vorgesehene Sonderberechtigung um einzelne Tickets freigeben zu können. |
| Mahnwesen (benötigt
  "Rechnungen") | Erstellung zu Mahnungen für überfällige Rechnungen. |
| Ticketadministration | Berechtigung Tickets zwischen Projekten zu verschieben oder zu löschen. |
| Ressourcenplanung | Planung von Mitarbeitern auf verschiedenen Projekten inkl. Auslastungsübersicht. |
| Crm Lead | Akquisebereich unter CRM>Leads um Anfragen verarbeiten zu können (empfohlen mit Angeboten). |
| Unternehmen Mitarbeiter | Zugriff auf sensible Mitarbeiterdaten wie z.B. Gehalt (wird über den Support freigegeben). |
| Projektplanung Gantt | Erstellung und Bearbeitung von Projektablaufplänen. |
| Ticketwert | Möglichkeit, einem Ticket zusätzlich zu einem Zeitkontingent auch einen festen Budgetwert zuzuweisen. Mit zusätzlicher Berechtigung für ‘Reporting’ > ‘Projektauswertung Detail’ werden diese den aktuell erfassten Stundenkosten gegenübergestellt. |
| Abwesenheitsanträge | Freigabe von eingehenden Abwesenheitsanträgen. **Zugriff Admin:** Dauer kann nachträglich bearbeitet werden. |
| Materialerfassung | Neben Zeiten können auch Materialeinsätze auf Projekte/Tickets gebucht werden. |
| Materialverwaltung | Anlage und Bearbeitung verfügbarer Materialien (inkl. Kosten). |
| Bestellwesen | Erstellung und Versand von Bestellanfragen und Beauftragungen an Lieferanten innerhalb eines Projekts.
**Eingeschränkter Zugriff:** Der Benutzer kann nur auf Bestellungen bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist zugreifen. |
| Arbeitsort Widget | Anzeige der Verfügbarkeit von Personen innerhalb des Unternehmens (Homeoffice, Abwesend, Büro). |
| E-Mails löschen | Information zu versendeten Mails löschen. |
| E-Mail Gesamtverlauf | Anzeige eines Verlaufs aller aus dem System versendeten Mails. |
| Inbox | Zugriff auf die Upload Inbox (BillBox) zur automatischen Eingangsrechnungsverarbeitung.
**Eingeschränkter Zugriff**: Der Benutzer kann nur selbst erstellte Dateien verarbeiten. |
| Stundenmanagement | Aufstellung von Einzelbuchungen von Stunden im Unternehmen.
**Eingeschränkter Zugriff:** Der Benutzer kann nur auf erfasste Zeiten bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist zugreifen. |
| Zeiten bearbeiten | Projekt, Phase, Ticket, Aufgabe, Kommentare, Leistung, Person und Abrechnung können verändert werden. Eingeschränkter Zugriff: Der Benutzer kann nur erfasste Zeiten bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist ändern. |
| Stunden ändern / löschen | Erfasste Stunden und Datum können geändert oder entfernt werden. Eingeschränkter Zugriff: Der Benutzer kann nur erfasste Zeiten bei denen er Projektverantwortlicher oder Projektmitverantwortlicher ist ändern oder löschen. |
| Eingangsrechnungen | Erfassung und Bearbeitung von Eingangsrechnungen in der Buchhaltung.
**Eingeschränkter Zugriff:** Der Benutzer kann nur auf von Ihm erstellte oder Ihm zugewiesene Eingangsrechnungen zugreifen.
**Wichtiger Hinweis**: Die Berechtigung ‘Eingangsrechnungen buchen’ ermöglicht unabhängig von dieser Berechtigung den Zugriff auf Eingangsrechnungen. |
| Eingangsrechnungen Storno aufheben | Stornierte Eingangsrechnungen wieder “aktivieren”. |
| Eingangsrechnungen löschen | Bestehende Eingangsrechnungen löschen. |
| Eingangsrechnungen buchen | Eingangsrechnungen als “Bezahlt” markieren.
**Eingeschränkter Zugriff**: Der Benutzer kann nur von Ihm erstellte oder Ihm zugewiesene Eingangsrechnungen buchen. |
| Kostenmanagement | Aufstellung aller erfassten Eingangsrechnungspositionen und die Möglichkeit die interne Verbuchung zu bearbeiten.
**Eingeschränkter Zugriff:** Es werden nur Kostenpositionen aus Projekten angezeigt, bei denen der Benutzer (Mit-)Verantwortlich ist. |
| Fremdkosten Projekte | Einsicht in den Bereich Kosten an einem Projekt zur Übersicht der dort erfassten Eingangsrechnungspositionen. Gültig für alle Projekte die nicht intern sind. Eingeschränkter Zugriff: Der Benutzer kann nur auf Belege von Ihm erstellten oder Ihm zugewiesenen Eingangsrechnungen zugreifen. |
| Fremdkosten interne Projekte | Einsicht in den Bereich Kosten an einem Projekt zur Übersicht der dort erfassten Eingangsrechnungspositionen. Gültig für alle Projekte die intern sind. Eingeschränkter Zugriff: Der Benutzer kann nur auf Belege von Ihm erstellten oder Ihm zugewiesenen Eingangsrechnungen zugreifen. |
| Fremdkosten Zuordnung | Benutzer ohne Zugriff auf Eingangsrechnungen können Phase, die Option ‘abrechenbar’ und die Auftragszuordnung einer Eingangsrechnungsposition direkt im Projektreiter ‘Kosten’ ändern. |
| Projektauswertung | Globale Auswertung zur Unternehmens-, Kunden-, und Projektkennzahlen wie Rohertrag, Stundenkosten, Projektergebnis. |
| Projektskizzen | Schnellanlage von Projektwerten ohne hinterlegte Kalkulation (z.B. Kosten in Höhe von X oder Umsatz in Höhe von Y). Erfordert ggf. eigene Aktivierung über den Support. |
| Projektauswertung Detail | Zusammenfassende Informationen (Summenbeträge, Status, Titel) zu Angeboten, Aufträgen, Rechnungen, Fremdkosten und Stundenkosten in der Detailansicht eines einzelnen Projekts. |
| Projektauswertung Detail Stunden | Diese Berechtigung greift nur wenn die ‘Projektauswertung Detail’ mit ‘kein Zugriff'eingeschränkt ist. Damit kann die Anzeige der geleisteten Stunden aktiviert werden ohne die wirtschaftliche Kalkulation anzuzeigen. |
| Quartalsauswertung Umsatz | Monats- und Quartalsweise Übersicht der geplanten und realisierten Umsätze im gesamten Unternehmen. Werte werden Monatlich nach Kunden und Projekten aufgeteilt dargestellt. |
| Quartalsauswertung Kosten | Monats- und Quartalsweise Übersicht der geplanten und angefallenen Kosten im gesamten Unternehmen. Werte werden Monatlich nach Lieferanten aufgeteilt dargestellt. |
| Quartalsauswertung Kosten Gehälter | Gehälter werden ausschließlich summiert, getrennt in regelmäßige monatliche Zahlungen und Sonderzahlungen, angeführt. Diese Berechtigung kann nur vom Poool Support verändert werden und steht in der Benutzersimulation nicht zur Verfügung. |
| Kostenauswertung Budget | Übersicht über den Stand der in den Einstellungen angelegten Budgets inkl. Entwicklung. |
| Kostenauswertung Kostenstellen | Übersicht der angefallenen Kosten nach Konstenstelle. |
| Kostenauswertung Kostenaufteilung | Übersicht der angefallenen Kosten nach Kostenaufteilung. |
| Kundenauswertung Top Kunden | Auswertung der Kennzahlen Projektvolumen, Rohertrag und Projektergebnis nach Kunden. |
| Kostenauswertung Top Lieferanten | Auswertung der Kosten summiert nach Lieferanten. |
| Zielauswertung | Übersicht über die Erreichung der angelegten Ziele. |
| Planner | Zugriff auf die Ressourcenplanung über die Schnellzugriffsleiste. |
| Freigaben Management | Der Benutzer kann eine Liste über alle offenen Freigaben einsehen. Eigene Freigaben werden einem Benutzer immer angezeigt. |
| Freigaben Administration | Der Benutzer kann alle Freigabeschritte selbst durchführen, ganze Freigaben zurücksetzen und deaktivieren. |
| Zahlungsverkehr / Transaktionen | Zugriff: Der Benutzer sieht alle Transaktionen Ihrer verknüpften Bankkonten. Eingeschränkter Zugriff: Der Benutzer kann nur Transaktionen sehen die eine erkannte oder mögliche Zuordnung zu einer Rechnung haben. Wichtig: Durch das automatische Zuordnungssystem können theoretisch sensible Transaktionen sichtbar werden. |
| Spesen und Spesenabrechnung | Erfassung und Einreichung von Spesenabrechnungen.
**Eingeschränkter Zugriff:** Der Benutzer kann nur selbst erstellte Spesen und Spesenabrechnungen einsehen und bearbeiten |
| Spesenabrechungen genehmigen | Um Spesenabrechnungen zu genehmigen braucht der Benutzer auch den Zugriff auf: Spesen und Spesenabrechnung, Eingangsrechnungen |
| Zeiträume hinzufügen / bearbeiten | In der Arbeitszeiterfassung können Buchungen direkt eingetragen und nachträglich bearbeitet werden. |
| Zeiträume mit Abwesenheitsantrag bearbeiten | Buchungen mit Antrag (Eintrag zu beantragten Zeiterfassungstypen wie z.B. Urlaub) können nachträglich bearbeitet und Stundenweise verlängert/verkürzt werden. |
| Zeiträume löschen | Gebuchte Einträge in der Zeiterfassung können nachträglich gelöscht werden. |
| Mitarbeiter Zeiterfassungsmanagement | Zugriff auf die Arbeitszeiterfassung anderer Mitarbeiter über der Bereich Unternehmen > Mitarbeiterzeiterfassung. |
| Inventarverwaltung | Lese- oder Lese- und Schreibzugriff auf die vergebenen und verfügbaren Inventarpositionen. |
| Inventar Protokolle | Möglichkeit zu einer Übergabe Protkolle zu erfassen und erstellen. |
| Inventar Kategorien | Lese- oder Lese- und Schreibzugriff auf die definierten Inventarkategorien und deren Eigenschaften. |
