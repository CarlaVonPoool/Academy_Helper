## Filtern von Listen

Eine **Liste** in Poool lässt sich in verschiedenen Bereichen **filtern**. Die Anwendung der **Filtereinstellungen** funktioniert in allen Poool-Bereichen gleich.

In unserem Beispiel filtern wir im Bereich Projektmanagement, welches Sie über `Projekte → Management` erreichen können.

Dort finden Sie die **Standardeinstellung** `> Filter: 1 Kriterium`. Wenn Sie auf die **Standardeinstellung** klicken, öffnet sich ein **Dropdown-Menü**. Aktuell finden Sie dort einige Projekte - Sie möchten allerdings nur jene **Projekte** sehen, welche den Tag (Kunde) "Neukunde", den Projektstatus "in Arbeit" sowie nicht den Verantwortlichen "Unser Alfred" hinterlegt haben.

Dafür klicken Sie auf `+ Filter` und **wählen** die gewünschten **Filter**. 

Wählen Sie folgende Filter aus und klicken auf `→ Anwenden`:

![filter-und-vorlagen-1.png](Filter%20&%20Vorlagen/filter-und-vorlagen-1.png)

Aus den unzähligen und **ungefilterten Projekten** werden nur mehr jene Projekte angezeigt, welche den **genannten Filtereinstellungen** entsprechen.

![filter-und-vorlagen-2.png](Filter%20&%20Vorlagen/filter-und-vorlagen-2.png)

Diese Listen können zusätzlich als **Excel-Datei** exportiert werden. Dazu klicken Sie unter den Filtereinstellungen auf `→ Export (xslx)`.

## Tab Filter

Im **Ticketmanagement**, welches Sie über `Tickets` erreichen, können Sie verschiedene **Tabs** anlegen. Wenn Sie auf das `+` in der **Tableiste** klicken, öffnet sich ein **Dropdown-Menü**, in dem Sie verschiedene Einstellungen vornehmen können. Vergeben Sie einen **Tab-Titel** sowie eine **Schrift-** und **Hintergrundfarbe** und nehmen Sie die gewünschten Filtereinstellungen vor. Anschließend klicken Sie auf `Speichern`.

![filter-und-vorlagen-3.png](Filter%20&%20Vorlagen/filter-und-vorlagen-3.png)

Es können beliebig viele **Tab-Filter** in den Bereichen Tickets, Kosten und Stundenmanagement **verwendet** werden.

## Vorlagen

In Poool können verschiedene **Vorlagen** erstellt werden, darunter auch eine Vorlage im Projektmanagement `Projekte → Management` . Diese übernimmt **automatisch** die **Standardeinstellungen**, wenn Sie die gewünschte Vorlage auswählen.

Im **Projektmanagement** befinden sich die “Vorlagen” neben der **Suchleiste**. Wenn Sie mit Mouseover auf “Vorlagen” klicken erscheinen zwei Zahnräder-Icons. 

![filter-und-vorlagen-4.png](Filter%20&%20Vorlagen/filter-und-vorlagen-4.png)

Dort können Sie unter `Vorlage bearbeiten` eine **neue Vorlage erstellen** und dieser Vorlage einen Titel vergeben. Zusätzlich können Sie auswählen, ob diese Vorlage Ihre **Standardvorlage** werden soll und, ob diese “Öffentlich” - also für weitere Mitarbeiter:Innen sichtbar sein soll.

![filter-und-vorlagen-5.png](Filter%20&%20Vorlagen/filter-und-vorlagen-5.png)

Immer, wenn Sie die Vorlage "BAT-Projekte" auswählen, werden sämtliche Filtereinstellungen verwendet, welche Sie zuvor getroffen haben.

![filter-und-vorlagen-6.png](Filter%20&%20Vorlagen/filter-und-vorlagen-6.png)
