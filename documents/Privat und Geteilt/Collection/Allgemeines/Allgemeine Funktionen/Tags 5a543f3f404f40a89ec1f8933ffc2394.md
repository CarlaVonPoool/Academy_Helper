# **Tags**

Tags sind ein zentrales **Feature** der Poool-Software. Sie dienen als **farbig hinterlegte Wörter** zur Kennzeichnung ("Flaggen") aller Elemente innerhalb der Poool-Umgebung. Die Verwendung von Tags ermöglicht es, bestimmte Informationen **schnell zu identifizieren** und zu klassifizieren, was zur **Verbesserung** der Effizienz und zur Optimierung der Workflows beiträgt.

### **Verwendung von Tags**

Tags können eine Vielzahl von Elementen in Poool kennzeichnen. Dazu gehören:

1. **Kontakte:** CRM-Leads, Firmen, Personen, Standorte
2. **Projekte:** Projekte, Projektphasen
3. **Ökonomie:** Angebote, Aufträge, Ausgangsrechnungen, Eingangsrechnungen
4. **Tickets:** Tickets, Aufgaben, Fragen, Kommentare
5. **Bestellwesen:** Bestellungen

### **Anpassung von Tags**

In den Tags kann sowohl die **Schriftfarbe** als auch die **Hintergrundfarbe** frei definiert werden. Dies ermöglicht eine hohe **Anpassungsfähigkeit** und gewährleistet, dass Ihre Tags den spezifischen **Anforderungen** Ihrer Arbeitsumgebung entsprechen.

### **Filterung durch Tags**

Dank unserer **Filterfunktionen** können Sie in jeder spezifischen Ansicht über die Tags filtern. Unsere Filter unterstützen sowohl ein **logisches UND** als auch ein **logisches ODER** bei der Verwendung von Tags, was Ihnen maximale **Flexibilität** bei der Durchsuchung Ihrer Daten bietet.

![tags-3.png](Tags/tags-3.png)

### **Anwendung von Tags**

Je nach Konfiguration können Sie Ihre **Tags** verwenden, um eine **Vielzahl von Funktionen** zu erfüllen. Sie können beispielsweise **Freigaben** abbilden, einen **Workflow** definieren oder einen anderen **spezifischen Anwendungsfall** darstellen.

### **Anwendungsbeispiele**

1. **Projektmanagement:** Tags können verwendet werden, um **verschiedene Phasen** eines Projekts zu markieren (z.B. "In Bearbeitung", "Überprüfung", "Abgeschlossen"). Dies ermöglicht eine klare **Visualisierung** des Projektfortschritts und hilft dabei, Prioritäten zu setzen.
2. **Kundenservice:** Tags können dazu dienen, die Kontakte nach verschiedenen **Kriterien** zu kategorisieren, z.B. nach Branchen (z.B. "Einzelhandel", "Fertigung") oder nach dem Status der Beziehung (z.B. "Lead", "Kunde", "Ex-Kunde").
3. **Aufgabenverwaltung:** Im Bereich Tickets können Tags helfen, den **Status** einer Aufgabe zu kennzeichnen (z.B. "Offen", "In Arbeit", "Gelöst") oder ihre **Dringlichkeit** anzugeben (z.B. "Niedrig", "Mittel", "Hoch").
4. **Finanzen:** Im Bereich Ökonomie können Sie Tags verwenden, um den **Status** von Rechnungen zu kennzeichnen (z.B. "Offen", "Bezahlt", "Überfällig") oder **verschiedene Arten** von Kosten zu kategorisieren (z.B. "Hardware", "Software", "Personal").

Durch die effektive Nutzung der Tags kann Poool dazu beitragen, dass Ihre **Workflows** reibungslos ablaufen, Ihre Projekte auf dem richtigen Weg bleiben und Ihre Organisation insgesamt effizienter arbeitet. Die Möglichkeiten sind nahezu unbegrenzt. Sie müssen nur herausfinden, wie Sie sie am besten nutzen können!
