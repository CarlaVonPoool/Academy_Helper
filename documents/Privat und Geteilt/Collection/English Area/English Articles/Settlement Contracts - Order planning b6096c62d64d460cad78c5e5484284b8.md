## Settlement with billing assistant

To create an invoice using the billing assistant, first create a new invoice in the desired project. After assigning invoice titles and the fields in the header of the invoice, start the billing assistant by clicking on the `magic wand-icon`.

Next, select the project that contains the order you want to bill. Then, click `order planning`.

In the next step, all orders available in the project are displayed, including their order scheduling items. All due order scheduling items are pre-selected for you. You can select or deselect individual items as required. The division is always done in internal and external contributions.

For the selected order scheduling items, you can then choose how they should appear on the invoice.

## Scheduling of orders individually

The schedule `individual option` inserts a separate item on the invoice for each selected schedule item. When selecting where to insert items, the billing assistant shows you all available groups of the invoice. If no group has been created yet, a new group is created automatically. If you select an existing group, the new items in the selected group are added.

Use `Add to Invoice` to add the items to your invoice as selected.

The description text is automatically generated from the text template stored in the billing assistant configuration.

## Contract planning in summary

If you want to group all selected order scheduling items into one row, select the scheduled order option `after the selection`. The sum of all selected order scheduling items is summarized on one line. Depending on the configuration of the billing assistant, all billed orders are listed in the item description.

Use `Add to Invoice` to add the items to your invoice as selected.

The description text is automatically generated from the text template stored in the billing assistant configuration.
