# Billing assistant

Last Updated: 4. August 2021

To make your billing as efficient as possible, you can set different presets for your billing texts in the billing wizard. The settings are divided according to the different billing options.

# Settings order planning

The settings for billing order schedules (e.g. flat rate) can be found under `Settings → Billing → Tab: Billing Assistant`. There you can store your standard texts for the different billing options.

## Groups & Items

If invoiced according to **order groups & items**, the original order groups and items are copied to the invoice. The texts to be copied are made in the corresponding settings area.

### Groups

When preparing the texts, it is possible to distinguish whether the order is with or without a previous offer. In the first case, data of the offer may also be shown on the invoice (e.g. invoicing according to the offer {OFFER_NUMBER} from {OFFER_DATE})

If the original texts are to be used for the job, the corresponding placeholders should be stored in all text fields.

### Positions

Similar to groups, positions can be distinguished whether there is an offer or not. The standard for settling positions is the adoption of the original description.

If no text should be copied, the placeholder `{EMPTY}` can be used.

## Scheduling of orders individually

If order planning is billed individually, 1 item is created on the invoice per selected order planning. The items to be created are grouped together in a separate group or can be added to an existing group at a later date.

The settings for the corresponding texts can be found here:

### One order

If 1 order is invoiced, the order title offers itself as a group title. In the item description of the order planning to be invoiced, it is possible to distinguish whether there was an offer or not. In the first case, you can refer to information from the offer when invoicing.

The order planning title is the name used in the order planning (e.g. 1st instalment).

### Multiple orders

If several orders are combined on one invoice, the title of the group is adjusted accordingly. For each selected order scheduling item, a position with the texts configured here is then created. Here, too, the offer information can be provided, if available.

## Order planning in summary

If order planning is billed in summary, all selected order planning items are bundled on one line in the invoice. The assignment of individual orders and their invoice totals takes place here (if at all desired) only in textual terms.

The settings can be found here:

### One order

If only one order is charged here, the presentation differs little from the one item per scheduling item (see above).
