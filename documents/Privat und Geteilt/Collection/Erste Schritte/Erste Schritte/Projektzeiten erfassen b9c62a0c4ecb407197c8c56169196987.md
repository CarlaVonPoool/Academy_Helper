# Projektzeiten erfassen

Poool trennt zwischen der reinen **Arbeitszeiterfassung** und der **Erfassung von Stunden auf Projekte**. Leistungsbezogene Stunden erfassen Sie, indem Sie ein Ziel (z. B. ein Projekt oder Ticket), die benötigte Zeit sowie die jeweilige Leistung angeben. 

Die erfassten Zeiten sind im **Stundenmanagement** einsehbar und können bei aufwandsbezogener Verrechnung direkt in eine neue Projektrechnung übernommen werden. Projektzeiten werden entweder direkt auf das Projekt oder ein darunterliegendes Element, wie eine Aufgabe oder ein Ticket, gebucht. In Poool können Projektzeiten auf zwei Wegen gebucht werden.

[https://vimeo.com/829716874](https://vimeo.com/829716874)

### **Methode 1:** Über das **Schnellzugriffsmenü**

1. Klicken Sie auf das **Uhr-Symbol** im Schnellzugriffsmenü, das sich auf der rechten Seite der Benutzeroberfläche befindet.
    
    ![projektzeiten-1.png](Projektzeiten%20erfassen/projektzeiten-1.png)
    
2. Wählen Sie ein **Ziel** für Ihre Zeiten aus, z.B. ein Projekt, ein Ticket oder eine Aufgabe.
3. Geben Sie die benötigte Zeit entweder **direkt** oder über die **inkrementale Erhöhung** durch die Buttons an.
4. Führen Sie die **Beschreibung der Tätigkeit** im Kommentarfeld genauer aus.
5. Wählen sie optional eine andere Funktion aus, ansonsten wird Ihre Standardfunktion gebucht. 
6. Speichern sie den Eintrag über `→ Zeit erfassen`

![projektzeiten-2.png](Projektzeiten%20erfassen/projektzeiten-2.png)

### **Methode 2:** Über die Projekt-Zeiterfassung

1. Navigieren Sie zum Reiter `Zeiterfassung`.
2. Klicken Sie auf `Projektzeit`, um eine neue Zeile für die **Erfassung von Projektzeiten** zu erstellen.

![image.png](Projektzeiten%20erfassen/image.png)

1. Wählen Sie **ein Ziel für** Ihre Zeiten aus, z.B. ein Projekt, ein Ticket oder Aufgabe.
2. Geben Sie die **benötigte Zeit** ein und klicken Sie auf `Speichern`, um die Zeit zu erfassen.

![projektzeiten-4.png](Projektzeiten%20erfassen/projektzeiten-4.png)

## Stundenmanagement

Die **gebuchten Zeiten** können im **Stundenmanagement** eingesehen werden. Navigieren Sie zu `Buchhaltung → Stundenmanagement`, um die **Übersicht** über alle **gebuchten Zeiten** zu erhalten.

![projektzeiten-5.png](Projektzeiten%20erfassen/projektzeiten-5.png)

![image.png](Projektzeiten%20erfassen/image%201.png)

Bei **aufwandsbezogener Verrechnung** können Sie die gebuchten Zeiten auf eine neue **Rechnung** im Projekt **verlinken**. Öffnen Sie das gewünschte Projekt und erstellen Sie eine neue Rechnung oder wählen Sie eine bestehende aus.

Über `Buchhaltung → Ausgangsrechnungen` gelangen Sie in die **Übersicht** Ihrer erstellten Rechnungen. Wählen Sie die **passende Rechnung** aus und verwenden den **Abrechnungsassistenten**, um die verrechenbaren Stunden zur Rechnung hinzuzufügen. Der Abrechnungsassistent hilft Ihnen dabei, die **erfassten Stunden** in die Rechnung zu **übernehmen**.

![projektzeiten-7.png](Projektzeiten%20erfassen/projektzeiten-7.png)

Überprüfen Sie die Rechnung, um sicherzustellen, dass **alle verrechenbaren Stunden** korrekt hinzugefügt wurden, und **speichern** Sie die Rechnung.

[Mehr Informationen, Stunden aus dem Stundenmanagement direkt abzurechnen, finden Sie hier.](../../Buchhaltung/Abrechnung/Stundenabrechnung%20direkt%20aus%20Stundenmanagement%20245015f5d93f807d8608dbf0dd020378.md)
