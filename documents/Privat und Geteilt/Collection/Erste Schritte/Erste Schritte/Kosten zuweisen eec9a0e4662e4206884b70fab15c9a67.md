# Kosten zuweisen

In Poool **weisen** Sie **Kosten** einer Eingangsrechnung einem Projekt oder Auftrag **zu**. Erstellen Sie dazu unter `Buchhaltung → Eingangsrechnungen` eine **neue Rechnung** und wählen das **Projekt** oder den **Auftrag**. Markieren Sie **Kosten** als `abrechenbar`, um zu sehen, welche dem Kunden **weiterverrechnet** werden. Kosten können auch einer **Kostenstelle**, einem **Budget** oder einer **Aufteilung** zugeordnet werden. In der Projektübersicht werden sie vom Rohertrag abgezogen.

[https://vimeo.com/829717036](https://vimeo.com/829717036)

1. Navigieren Sie zu `Buchhaltung → Eingangsrechnung`.

![kosten-zuweisen-1.png](Kosten%20zuweisen/kosten-zuweisen-1.png)

1. Erstellen Sie eine neue **Eingangsrechnung**, indem Sie auf `+ Eingangsrechnung` klicken.

![kosten-zuweisen-2.png](Kosten%20zuweisen/kosten-zuweisen-2.png)

1. Füllen Sie die **erforderlichen Informationen** für die Eingangsrechnung aus, wie z.B. Lieferant, Rechnungsnummer und Rechnungsdatum.

![kosten-zuweisen-4.png](Kosten%20zuweisen/kosten-zuweisen-4.png)

1. Wählen Sie bei der **Leistungszuweisung** das entsprechende **Projekt** oder den **Auftrag** aus, dem die **Kosten zugeordnet** werden sollen.

![kosten-zuweisen-3.png](Kosten%20zuweisen/kosten-zuweisen-3.png)

1. Markieren Sie die **Kosten** als `abrechenbar`, wenn diese dem **Kunden weiterverrechnet** werden sollen.
2. Optional können Sie die Kosten einer **Kostenstelle**, einem **Budget** oder einer **Aufteilung** **zuweisen**.
3. **Speichern** Sie die Eingangsrechnung, indem Sie auf `Speichern` klicken.
4. In der Projektübersicht werden die **zugewiesenen Kosten** direkt vom **Rohertrag** abgezogen. Navigieren Sie zur **Projektübersicht**, um den aktualisierten Rohertrag zu überprüfen.

<aside>
ℹ️ Mit diesen Schritten haben Sie erfolgreich **Kosten** einer Eingangsrechnung einem Projekt oder Auftrag in Poool **zugewiesen**. Das Zuweisen von Kosten ermöglicht eine bessere **Kostenkontrolle** und hilft Ihnen, den Rohertrag Ihrer Projekte und Aufträge genauer zu überwachen.

</aside>
