# Dashboard einrichten

Ein Dashboard in Poool unterstützt Sie bei der individuellen Gestaltung Ihrer Arbeitsumgebung und verschafft Ihnen einen schnellen Überblick über wichtige Informationen. Durch einen Doppelklick können Sie Widgets hinzufügen und das Dashboard so an Ihre Bedürfnisse anpassen. Widgets wie **„Offene Leads“** oder **„Offene Aufträge“** ermöglichen einen direkten Einblick in laufende Projekte. Auf diese Weise steigern Sie Übersicht, Produktivität und Effizienz.

Um ein Dashboard anzulegen, klicken Sie auf die Schaltfläche `+ Dashboard` um Ihr erstes Dashboard anzulegen.

![dashboard-3.png](Dashboard%20einrichten/dashboard-3.png)

Es gibt zwei Möglichkeiten, Widgets zu Ihrem Dashboard hinzuzufügen: 

Klicken Sie auf  `+ Widget`oder doppelklicken Sie auf den Hintergrund, um die Widget-Übersicht zu öffnen und Ihre gewünschten Widgets auszuwählen. Die Position der Widgets können Sie anschließend ganz einfach per **Drag & Drop** anpassen.

Weitere Informationen zur Einrichtung des Dashboards finden Sie [hier](../../Setup%20&%20Konfiguration/Poool%20Setup/Dashboard%2020400be653b347f3a29ab374b4cb154f.md).
