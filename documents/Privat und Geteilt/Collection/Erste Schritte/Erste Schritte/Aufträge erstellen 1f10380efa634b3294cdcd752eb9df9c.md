# **Aufträge erstellen**

Ein Auftrag in Poool ist die Baasis für Zeiterfassung, Abrechnung und Controlling. In Poool werden Angebote “beauftragt” und automatisch in Aufträge umformuliert. Genauso aber lassen sich Aufträge auch ohne vorheriges Angebot erstellen. 

### ✅ Voraussetzungen

- Projekt ist angelegt
- Dokumentenvorlage ist eingerichtet
- Angebot liegt vor oder sie kennen die zu erbringenden Leistungen

[https://vimeo.com/829716756?share=copy](https://vimeo.com/829716756?share=copy)

## **📌 Prozessbeschreibung: Schritt-für-Schritt Anleitung**

### Variante 1 - Auftrag aus Angebot erstellen

In Poool können Sie **Angebote vollständig** oder einzelne Positionsgruppen **beauftragen**. Optionale Positionsgruppen bieten **Flexibilität**, ohne die Gesamtsumme unnötig zu erhöhen. Kunden entscheiden, ob sie diese berücksichtigen möchten. So **passen** Sie das Angebot **an** Projektanforderungen und Budget an.

- Navigieren Sie zum **Angebotsbereich** unter `Projekte → Angebote` in der Software und öffnen Sie das gewünschte Angebot.
    
    ![beauftragung.png](Auftr%C3%A4ge%20erstellen/beauftragung.png)
    
- Prüfen Sie das Angebot und achten Sie auf die enthaltenen **Positionsgruppen**, einschließlich *optionaler* Positionsgruppen.
- Um den **Beauftragungsprozess** zu starten, stellen Sie den Status auf `Beauftragt` um.

![beauftragung-2 – 1.png](Auftr%C3%A4ge%20erstellen/beauftragung-2__1.png)

- Dies öffnet die **Beauftragung**, in der Sie wählen können, ob Sie das **gesamte Angebot** beauftragen, **optionale Positionen** mitnehmen oder weglassen und ob Sie das Angebot eventuell sogar nur teilbeauftragen möchten.

![beauftragung-3.png](Auftr%C3%A4ge%20erstellen/beauftragung-3.png)

- Entscheiden Sie sich für die **gewünschte Beauftragungsoption**: Gesamtbeauftragung (OK), Teilbeauftragung (Erweiterte Gruppenbeauftragung) oder die Berücksichtigung optionaler Positionsgruppen.
- Bestätigen Sie die Beauftragung und speichern Sie den Auftrag, indem Sie auf die Schaltfläche `→ OK` klicken.

### Variante 2 - Auftrag ohne Angebot direkt anlegen

Manchmal brauchen Sie kein formales Angebot – z.B. bei Stammkunden oder internen Projekten.

- Navigieren Sie zum **Angebotsbereich** unter `Projekte → Angebote` in der Software und öffnen Sie das gewünschte Angebot.
- Wechseln Sie zum Reiter `Aufträge`
- Klicken Sie auf `+Auftrag`
- Geben Sie die Auftragsinformationen ein
    - Titel, Nummer, Datum
- Fügen Sie die Leistungen, Mengen und Stunden aus der Preisliste hinzu
- Speichern Sie den Auftrag

### Auftragsplanung

In den Aufträgen können Sie dann weitere Einstellungen treffen, beispielsweise in der Auftragsplanung, die die Abrechnungsformen definiert. Mehr dazu lesen Sie [hier.](https://www.notion.so/56cbadaa128840478691330fc42f0b15?pvs=21) 

![image.png](Auftr%C3%A4ge%20erstellen/image.png)

Ebenso können Sie direkt aus dem Auftrag Tickets generieren über den Button `→ Zuweisen` .
