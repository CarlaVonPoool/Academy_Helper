# Auftragsverrechnung

In Poool gibt es **pauschale** und **aufwandsbasierte** Auftragsverrechnung. Sie können **Teilverrechnungen** und **Schlussrechnungen** planen.  Der Status eines Auftrags (offen/abgerechnet) speist sich aus der Auftragsplanung und wird automatisch angepasst. Es gibt zwei Wege, einen Auftrag abzurechnen bzw. aus diesem eine Rechnung zu erstellen.

[https://vimeo.com/829716816](https://vimeo.com/829716816)

### Erste Variante: Abrechnung über Auftragsplanung

1. Navigieren Sie zum **Auftragsbereich** in der Software unter `Projekte → Aufträge` und öffnen Sie den gewünschten Auftrag.

![auftragsverrechnung-1.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/auftragsverrechnung-1.png)

1. Achten Sie darauf, dass die Stundenabrechnung richtig hinterlegt ist: 
    1. Auftrag inkludiert Stunden (pauschal) 
    2. Stunden direkt abrechnen (aufwandsbasiert)
    3. Stunden nicht abrechnen

![auftragsverrechnung-2.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/auftragsverrechnung-2.png)

1. Klicken Sie nun auf die **Auftragsplanung**. Hier legen Sie fest, wie der Auftrag für die Abrechnung geplant werden soll. Im Planungsmodus können Sie beispielsweise einstellen, dass der Auftrag über eine An- und Restzahlung (z.B. 50/50) oder nach Gruppen abgerechnet wird. Belassen Sie die Einstellung auf **„automatisch“**, sind keine weiteren Anpassungen erforderlich.

![image.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/image.png)

Die **Abrechnungspläne** können Sie individuell hinterlegen. Lesen Sie [hier](../../Buchhaltung/Abrechnung/Abrechnungspl%C3%A4ne%20e1d4e4610dc746cc95b181f6f3f6ce08.md) mehr dazu.

1. Klicken Sie weiter auf das blaue `+ Rechnungen erstellen` und es erscheint ein kleines Fenster, in dem Sie ein Rechnungsdatum eingeben und die Positionsübernahme bestimmen. Hier wählen Sie, ob Sie die Summe auf eine Positionszeile zusammengefasst haben möchten oder die originalen Auftragspositionen übernommen werden sollen.

![image.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/image%201.png)

1. Sie werden in den Entwurf einer Rechnung navigiert. Hier können Sie noch fehlende Informationen ergänzen und mit dem **Abrechnungsassistent** (Zauberstab) noch fehlende Stunden zubuchen.

![image.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/image%202.png)

[Mehr zum Abrechnungsassistenten lesen Sie hier.](../../Buchhaltung/Abrechnung/Abrechnungsassistent%2026a7806fc295433b9785e5072b977d86.md) 

1. Nachdem Sie den **Rechnungsentwurf** geprüft haben, klicken Sie auf `abrechnen`, um der Rechnung eine **Rechnungsnummer** zuzuweisen.

![image.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/image%203.png)

1. **Speichern** Sie die erstellte Rechnung, indem Sie auf die Schaltfläche `Speichern` klicken.
2. Nun können Sie die Rechnung als PDF exportieren, drucken oder per E-Mail an den Kunden senden, indem Sie die entsprechenden Optionen im Reiter `Dokumente` der Kalkulation auswählen.

### Zweite Variante: Abrechnung über Abrechnungsassistent

1. Um nun eine **Abrechnung** für dieses Projekt zu erstellen, wechseln Sie in den **Rechnungsbereich** des Projektes (`Reiter: Rechnungen`).

![auftragsverrechnung-3.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/auftragsverrechnung-3.png)

1. Erstellen Sie über `+ Ausgangsrechnung` eine neue Rechnung und tragen dort einen Rechnungstitel ein.
2. Verwenden Sie den **Abrechnungsassistenten** in der Rechnungskalkulation, um verrechenbare **Stunden** und **Fremdleistungen oder Positionen über die Auftragsplanung** zur Rechnung hinzuzufügen. Sie können den Abrechnungsassistenten beliebig oft verwenden, um Dinge zur Rechnung hinzuzufügen. 

![auftragsverrechnung-4.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/auftragsverrechnung-4.png)

1. Optional können Sie eine Rechnung komplett **individuell erstellen** und nicht nur die Leistungen aus der Preisliste verwenden. Passen Sie die Rechnung nach Ihren **Bedürfnissen** an.
2. Überprüfen Sie den **Rechnungsentwurf** und stellen Sie sicher, dass alle erforderlichen **Informationen korrekt** sind.
3. Nachdem Sie den **Rechnungsentwurf** geprüft haben, klicken Sie auf `abrechnen`, um der Rechnung eine **Rechnungsnummer** zuzuweisen.

![image.png](../../Setup%20&%20Konfiguration%202620-dc4d/%E2%80%9CErste%20Schritte%E2%80%9D/Auftragsverrechnung/image%204.png)

1. **Speichern** Sie die erstellte Rechnung, indem Sie auf die Schaltfläche `Speichern` klicken.
2. Nun können Sie die Rechnung als PDF exportieren, drucken oder per E-Mail an den Kunden senden, indem Sie die entsprechenden Optionen im Reiter `Dokumente` der Kalkulation auswählen.

<aside>
💡

Die Variante über den **Abrechnungsassistent** hilft bei verrechenbaren Stunden und Fremdleistungen. 

</aside>
