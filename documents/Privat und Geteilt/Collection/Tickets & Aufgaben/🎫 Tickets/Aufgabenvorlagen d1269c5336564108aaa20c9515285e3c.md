## Aufgaben**vorlagen**

Mit Hilfe der Aufgabenvorlagen können Sie Ihren Ticketvorlagen einfach und schnell Aufgaben und Todo’s hinzufügen.

### Aufgabenvorlage erstellen

Um eine neue Aufgabenvorlage zu erstellen, navigieren Sie zu `Einstellungen → Tickets`. Dort wechseln Sie auf den Raster `Aufgabenvorlagengruppen`. Durch `+ Aufgabenvorlagengruppe` können Sie eine neue Vorlagengruppe erstellen.

![Untitled](Aufgabenvorlagen/Untitled.png)

Nach der Erstellung der Vorlagengruppen können Sie die eigentliche Vorlage erstellen. Wechseln Sie dazu auf den Raster `Aufgabenvorlagen` und drücken Sie auf `+ Aufgabenvorlagen`.

![Untitled](Aufgabenvorlagen/Untitled%201.png)

Es öffnet sich ein Fenster in dem Sie nun die gewünschten Daten für Ihre Vorlage eingeben können. Im ersten Schritt müssen Sie sich eine zuvor angelegte Gruppe für die Vorlage aussuchen. Im nächsten Schritt geben Sie den gewünschten Titel ein. Optional können Sie noch eine kurze Beschreibung hinzufügen. Sie können der Aufgabenvorlage auch noch weitere Informationen wie Tags oder einen Verantwortlichen geben. Auch der Start- und Endtermin sowie ein Zeitkontingent kann der Aufgabe mitgegeben werden. “Start in” bedeutet hierbei, wann die Aufgabe gestartet werden soll und “Termin in” ist immer die Laufzeit ab dem Start. Wenn Sie alle relevanten Informationen ausgefüllt haben, die die Aufgabenvorlage haben soll, können Sie in der Aufgabe noch Todo’s anlegen. 

![Untitled](Aufgabenvorlagen/Untitled%202.png)

Um ein Todo anzulegen, drücken Sie in der Aufgabe auf `+ Todo` und geben Sie dem Todo einen Titel. Mit den Pfeilen neben dem Titel können Sie die Todo’s in Ihre gewünschte Reihenfolge bringen. Ganz rechts können Sie das Todo wieder löschen.

![Untitled](Aufgabenvorlagen/Untitled%203.png)

Als letzten Schritt müssen Sie noch `Speichern` drücken um die Vorlage inkl. Todo’s abzuspeichern.

### Aufgabe einer Ticketvorlage hinzufügen

Um eine Aufgabe einem Ticket hinzuzufügen, wechseln Sie wieder auf den Raster `Ticketvorlagen`. Öffnen Sie die Ticketvorlage, der Sie die Aufgabe hinzufügen wollen oder erstellen Sie eine neue Ticketvorlage. Durch das Drücken von `+ Aufgabe` öffnet sich ein Fenster in dem alle angelegten Aufgaben erscheinen. Wählen Sie eine oder mehrere Aufgaben die Sie dem Ticket hinzufügen wollen und drücken Sie auf `+ Hinzufügen`, um die Aufgabe dem Ticket hinzuzufügen.

![Untitled](Aufgabenvorlagen/Untitled%204.png)

Auf hier können Sie wieder auf der Rechten Seite die Reihenfolge der Aufgaben ändern oder eine Aufgabe löschen.

![Untitled](Aufgabenvorlagen/Untitled%205.png)

Durch `Speichern` wird die Ticketvorlage mit den dazugehörigen Aufgaben und Todo’s gespeichert.
