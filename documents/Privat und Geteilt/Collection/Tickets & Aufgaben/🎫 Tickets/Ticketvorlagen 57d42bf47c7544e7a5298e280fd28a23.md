## **Ticketvorlagen**

Ticketvorlagen sollen Ihnen dabei helfen wiederkehrende Ticketpakete schnell zu erstellen. 

### Ticketvorlage erstellen

Um eine neue Ticketvorlage zu erstellen, navigieren Sie zu `Einstellungen → Tickets`. Dort wechseln Sie auf den Raster `Ticketvorlagengruppen`. Durch `+ Ticketvorlagengruppe` können Sie eine neue Vorlagengruppe erstellen.

![Untitled](Ticketvorlagen/Untitled.png)

Nach der Erstellung der Vorlagengruppen können Sie die eigentliche Vorlage erstellen. Wechseln Sie dazu auf der Raster `Ticketvorlagen` und drücken Sie auf `+ Ticketvorlage`.

![Untitled](Ticketvorlagen/Untitled%201.png)

Es öffnet sich ein Fenster in dem Sie nun die gewünschten Daten für Ihre Vorlage eingeben können. Im ersten Schritt müssen Sie sich eine zuvor angelegte Gruppe für die Vorlage aussuchen. Im nächsten Schritt geben Sie den gewünschten Titel ein. Optional können Sie noch eine kurze Beschreibung hinzufügen. Sie können der Ticketvorlage auch noch weitere Informationen wie Tags, Prioritäten, einen Status oder einen Verantwortlichen geben. Auch der Start- und Endtermin sowie ein Zeitkontingent kann dem Ticket mitgegeben werden. “Start in” bedeutet hierbei, wann das Ticket gestartet werden soll und “Termin in” ist immer die Laufzeit ab dem Start. 

Wenn Sie alle relevanten Informationen ausgefüllt haben, die die Ticketvorlage haben soll, können Sie dem Ticket noch Todo’s oder Aufgaben hinzufügen. Mehr dazu finden Sie im Guide zu Aufgabenvorlagen.

Als letzten Schritt müssen Sie noch `Speichern` drücken um die Vorlage abzuspeichern.

![Untitled](Ticketvorlagen/Untitled%202.png)

Sowohl Ticketvorlagengruppen als auch Ticketvorlagen können mit dem Pfeil auf der rechten Seite verschoben werden, damit die gewünschte Reihenfolge besteht.

### Ticket aus Vorlage erstellen

Um ein Ticket anzulegen, dass Sie zuvor als Vorlage angelegt haben, navigieren Sie im Menü zu Tickets. Setzen Sie die Checkbox bei *aus Vorlage* und drücken Sie auf `+ Ticket`.

![Untitled](Ticketvorlagen/Untitled%203.png)

Es öffnet sich ein Fenster mit allen von Ihnen angelegten Vorlagen. Sie können oben den Nummernkreislauf sowie das Projekt in dem Sie das Ticket anlegen wollen, wählen. Sobald Sie das Zielprojekt gewählt haben, können Sie das Ticket auch einer gewünschten Phase oder einem Auftrag zuordnen. Im nächsten Schritt suchen Sie sich die Vorlage die Sie verwenden möchten. Setzen Sie bei der Vorlage auf der linken Seite den Haken um sie auszuwählen. Sie sehen hier auch einen Start- und Endtermin des Ticket - diesen können Sie nach belieben anpassen. Sie können auch mehrere Vorlagen oder ganze Vorlagengruppen auswählen, die Sie erstellen wollen. Haben Sie alle gewünschten Vorlagen ausgewählt, drücken Sie unten auf den Button `+ Hinzufügen` um die Vorlagen zu erstellen. 

![Untitled](Ticketvorlagen/Untitled%204.png)

Nach dem Hinzufügen wechselt Poool in die Ticketansicht des gewählten Projektes. Hier können Sie Ihre Tickets, wenn gewünscht, weiter bearbeiten und Informationen hinzufügen.

Dieser Prozess funktioniert auch wenn Sie sich direkt in einem bestimmten Projekt befinden. Der einzige Unterschied hierbei ist, dass das Projekt bereits festgelegt ist und nicht mehr geändert werden kann.
