# Preisliste erstellen

---

Navigieren Sie zu `Einstellungen → Preise & Funktionen`. Im ersten Schritt müssen Sie im Raster `Funktionsgruppen` ihre Funktionsgruppen anlegen. Wenn Sie das erledigt haben, können Sie ihre einzelnen Funktionen anlegen. Dazu gehen Sie zum Raster `Funktionen` und fügen mit `+ Funktion` eine neue Funktion hinzu. Im letzten Schritt können Sie im Raster Preislisten durch `+ Preisliste` eine neue Preisliste anlegen oder die ergänzten Funktionen einer bereits bestehenden Preisliste hinzufügen. 

![Untitled](Wie%20wird%20eine%20Preisliste%20erstellt/Untitled.png)

<aside>
ℹ️

**Funktionsgruppen**: Funktionsgruppen fassen ähnliche oder zusammengehörige Funktionen in Kategorien zusammen Sie sind die übergeordneten Kategorien, die bestimmte Themen oder Leistungsbereiche abdecken.

**Funktionen**: Funktionen sind die eigentlichen spezifischen Merkmale oder Werkzeuge innerhalb einer Funktionsgruppe, die eine bestimmte Aufgabe erfüllen. Sie sind detaillierter als die Funktionsgruppen und beschreiben die genauen Produkte oder Dienste die angeboten werden.

</aside>
