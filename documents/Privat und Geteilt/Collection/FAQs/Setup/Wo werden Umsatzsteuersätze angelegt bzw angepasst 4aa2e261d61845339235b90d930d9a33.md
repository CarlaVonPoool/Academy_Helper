Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

# Umsatzsteuersätze ändern

---

Dazu navigieren Sie zu `System → Umsatzsteuer`. Durch `+ Umsatzsteuer` können Sie neue Umsatzsteuersätze anlegen. Durch das Klicken auf den gewünschten Umsatzsteuersatz können Sie bestehende Umsatzsteuersätze verändern. 

<aside>
❗ Bitte beachten Sie, dass die Anpassung von Umsatzsteuersätzen Auswirkungen auf bestehende Kalkulationen haben kann.

</aside>
