# Urlaubsanträge löschen

---

<aside>
ℹ️

Urlaubsanträge können nicht vom Mitarbeiter selbst wieder gelöscht werden. Der Verantwortliche kann den Urlaubsantrag jederzeit auf storniert setzen. Somit wird dem Mitarbeiter kein Urlaubstag von seinem Kontingent abgezogen.

</aside>

Wenn Sie einen Urlaubsantrag löschen bzw. stornieren wollen, navigieren Sie im Menü zu `Unternehmen → Abwesenheiten`. Wechseln Sie zu den genehmigten Abwesenheiten und wählen Sie den Antrag, den Sie stornieren wollen. Öffnen Sie den Antrag und ändern Sie den Status von `→ Genehmigt` auf `→ Storniert` und speichern Sie den Antrag.

![image.png](Wie%20k%C3%B6nnen%20Urlaubsantr%C3%A4ge%20wieder%20gel%C3%B6scht%20werden/image.png)

<aside>
ℹ️

Wenn das Feld `→ Storniert` bei Ihnen ausgegraut ist, haben Sie nicht die entsprechenden Rechte, um einen Antrag zu stornieren. Wenden Sie sich an Ihren Vorgesetzten oder an den Poool-Support.

</aside>
