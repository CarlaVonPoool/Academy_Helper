# Passwort ändern / zurücksetzen

Das Passwort eines Benutzers kann auf drei verschiedene Wege geändert werden.

## Zurücksetzen über die Login Seite

Am unteren Ende der Login Seite befindet sich ein Link zum Zurücksetzten des Passworts. Das Passwort wird durch den folgenden Ablauf zurückgesetzt.

![image.png](Wie%20kann%20das%20Passwort%20ge%C3%A4ndert%20zur%C3%BCckgesetzt%20werde/image.png)

1. Link “Passwort vergessen”
2. E-Mail Adresse eingeben
3. Sofern die E-Mail-Adresse registriert ist, wird Ihnen ein Link zum Zurücksetzen des Passworts an ihre E-Mail Adresse gesendet 
4. Link in der E-Mail öffenen
5. Neues Passwort vergeben 
6. Login mit neuem Passwort durchführen

<aside>
📌

Bitte nicht vergessen, das Passwort im Passwortmanager zu aktualisieren.

</aside>

## Kontoeinstellungen (angemeldeter User)

Das Poool-Passwort wird über das Zahnrad-Icon (*neben dem Profilbild*) unter den Kontoeinstellungen geändert.

## Benutzerverwaltung (angemeldeter Administrator)

Der Poool Administrator kann unter `Benutzer → Verwaltung → Name des Mitarbeiters` das Passwort zurücksetzen und ein neues Passwort vergeben oder einen Wiederherstellungslink an den Benutzer versenden.
