# Wie wird ein User auf inaktiv gesetzt?

Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

# User inaktiv setzen

---

Dazu gehen Sie zu `Benutzer → Verwaltung` und wählen den gewünschten Mitarbeiter aus, den Sie deaktivieren wollen. Öffnen Sie den Mitarbeiter und entfernen Sie den Haken im Kästchen `Aktiv`. Nun ist der Mitarbeiter inaktiv und kann sich nicht mehr ins System einloggen.
