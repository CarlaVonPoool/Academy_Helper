# Können für unterschiedliche Mitarbeiter unterschiedliche Feiertagskalender angegeben werden?

Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

Unter `Einstellungen → Zeiterfassung` können Sie den Standard Feiertagskalender einstellen. 

![Untitled](K%C3%B6nnen%20f%C3%BCr%20unterschiedliche%20Mitarbeiter%20unterschie/Untitled.png)

Wenn Sie für einen bestimmten Mitarbeiter einen anderen Feiertagskalender hinterlegen wollen, weil dieser beispielsweise von wo anders aus arbeitet, navigieren Sie zu `Unternehmen → Mitarbeiter`. Dort können Sie den Feiertagskalender individuell anpassen.

![Untitled](K%C3%B6nnen%20f%C3%BCr%20unterschiedliche%20Mitarbeiter%20unterschie/Untitled%201.png)
