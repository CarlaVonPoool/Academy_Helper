# Krankmeldung hochladen

---

Um eine Krankmeldung in poool hochzuladen, gibt es zwei Möglichkeiten. Die Krankmeldung kann entweder vom Mitarbeiter hochgeladen werden, wenn der Antrag gestellt wird (Beim Widget Persönliche Infos mit `+ Abwesenheit`) .

![Untitled](Wo%20kann%20eine%20Krankmeldung%20hochgeladen%20werden/Untitled.png)

Oder von demjenigen der die Abwesenheiten bestätigt (`Unternehmen → Abwesenheiten`). Dort kann auch von dieser Person eine Datei hinzugefügt werden.

![Untitled](Wo%20kann%20eine%20Krankmeldung%20hochgeladen%20werden/Untitled%201.png)
