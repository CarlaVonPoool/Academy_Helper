# Wo kann ein Antrag auf Urlaub, Sonderurlaub oder Krankenstand gestellt werden?

Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

# Anträge stellen

---

Mit einem Doppelklick auf das Dashboard können Sie unterschiedliche Widgets hinzufügen. Für den Urlaubs-, Sonderurlaubs- oder Krankenstandsantrag benötigen Sie das Widget `Persönliche Infos`.

![Untitled](Wo%20kann%20ein%20Antrag%20auf%20Urlaub,%20Sonderurlaub%20oder%20K/Untitled.png)

Auf diesem Widget finden Sie einen Button mit der Aufschrift `+ Abwesenheiten`. Mit dem Klick auf den Button öffnet sich ein Fenster in dem Sie den Antrag stellen können.

Der Antrag muss dann noch vom Verantwortlichen genehmigt werden.
