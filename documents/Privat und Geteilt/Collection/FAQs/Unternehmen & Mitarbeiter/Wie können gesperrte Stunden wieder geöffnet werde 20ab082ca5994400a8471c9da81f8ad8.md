# Gesperrte Stunden wieder öffnen

---

Ein festgeschriebener Tag kann von der Geschäftsführung bzw. von einem Admin wieder geöffnet werden. Dazu müssen Sie unter `Benutzer → Verwaltung` den gewünschten Benutzer simulieren. Im Benutzer selbst, können Sie dann zur `Zeiterfassung` navigieren und der gewünschte Tag kann mit Klick auf das Schloss wieder geöffnet werden.
