# Nachträglich Zeiten erfassen

---

Zeiten können selbstverständlich auch nachträglich erfasst werden. Dafür klicken Sie auf den Reiter `Zeiterfassung` und wählen den gewünschten Tag. Dort können Sie durch `+ Projekterfassung` bzw. `+ interne Tätigkeiten` an dem gewünschten Tag Zeiten erfassen.

 

<aside>
ℹ️ Wichtig ist, dass das Schloss geöffnet ist. Wenn das Schloss gesperrt ist, können Sie keine Zeiten mehr erfassen - der Admin oder der Support kann Ihnen diesen Tag jederzeit gerne wieder öffnen.

</aside>
