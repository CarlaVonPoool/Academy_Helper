# Wo kann der Verantwortliche für die Abwesenheiten geändert werden?

Last Updated: 30. November 2023
Published: Yes
Suggested: No

### Globale Einstellung:

Unter `Einstellungen → Zeiterfassung` kann unter “*Abwesenheiten*” ein Verantwortlicher für die Bearbeitung der Abwesenheiten für alle Benutzer ausgewählt werden.

![](../Unternehmen%20&%20Mitarbeiter/Wo%20kann%20der%20Verantwortliche%20f%C3%BCr%20die%20Abwesenheiten%20/Untitled.png)

### Benutzerdefinierte Einstellung:

Unter `Unternehmen - Mitarbeiter - Max Mustermann` kann für Max Mustermann ausgewählt werden, ob die **globalen Einstellungen** oder ob **benutzerdefinierte Einstellungen** verwendet werden sollen. In diesem Beispiel ist für die Abwesenheiten von Max Mustermann nun Daniel Düsentrieb verantwortlich.
