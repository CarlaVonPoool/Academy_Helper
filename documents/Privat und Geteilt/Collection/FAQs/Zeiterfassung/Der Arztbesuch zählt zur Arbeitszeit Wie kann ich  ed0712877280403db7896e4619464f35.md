# Der Arztbesuch zählt zur Arbeitszeit. Wie kann ich das umstellen?

Published: Yes
Suggested: No

Unter **Einstellungen-Zeiterfassung-Zeiterfassungstypen** finden Sie die Konfiguration zu den Typen. 

Wählen Sie **Arztbesuch** und ändern Sie den **Basistyp** auf **Sonstige Abwesenheit**. Somit wird ein Arztbesuch nicht mehr als Arbeitszeit gerechnet.
