# Ticket bearbeiten

Für das Bearbeiten von Tickets kann es unterschiedliche Berechtigungen geben

---

### Option 1:

Nur der Ersteller eines Tickets, kann das Ticket auch bearbeiten. Das ist die Standardeinstellung.

### Option 2:

Der Ersteller, der Verantwortliche und der Mitverantwortliche können das Ticket bearbeiten. 

### Option 3:

Alle können das Ticket bearbeiten.

<aside>
ℹ️

Diese Einstellungen können vom Admin unter `Einstellungen → Tickets` angepasst werden.

</aside>
