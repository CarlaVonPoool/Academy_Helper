# Ticket anlegen

---

## 1. Möglichkeit

Oben in der Menüleiste finden Sie das Menü für `Tickets`. In diesem Menü finden Sie einen Button mit `+ Tickets`. Wenn Sie diesen Button drücken, öffnet sich ein neues Ticket, wo alle Informationen erfasst werden können.

## 2. Möglichkeit

Auf der rechten Seite in der Schnellmenü-Leiste finden Sie ein großes Plus. Mit diesem Plus können Sie einige oft benötigte Funktionen aufrufen. Darunter auch ein Lead. Wenn Sie auf das `+` und dann auf `Ticket` drücken, öffnet sich wieder das Fenster in dem alle erforderlichen Funktionen eingegeben werden können.
