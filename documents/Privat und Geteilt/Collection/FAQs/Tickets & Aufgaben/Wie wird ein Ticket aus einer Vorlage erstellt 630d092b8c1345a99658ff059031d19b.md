# Ticket aus Vorlage erstellen

---

Navigieren Sie im Menü zu `Tickets`. Setzen Sie den Haken bei “aus Vorlage” und drücken Sie `+ Ticket`.

![Untitled](Wie%20wird%20ein%20Ticket%20aus%20einer%20Vorlage%20erstellt/Untitled.png)

Es öffnet sich ein Menü. In diesem Menü müssen Sie das gewünschte Projekt wählen, in dem Sie das Ticket anlegen wollen und das gewünschte Ticket auf der linken Seite auswählen.

![Untitled](Wie%20wird%20ein%20Ticket%20aus%20einer%20Vorlage%20erstellt/Untitled%201.png)

Durch `+ Hinzufügen` wird das neue Ticket erstellt.
