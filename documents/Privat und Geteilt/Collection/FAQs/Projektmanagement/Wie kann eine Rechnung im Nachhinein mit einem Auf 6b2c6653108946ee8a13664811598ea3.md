# Rechnung im Nachhinein mit Auftrag verknüpfen

---

Dazu suchen Sie sich die gewünschte Rechnung die Sie mit einem Auftrag verknüpfen wollen. Klappen Sie mit dem Bearbeiten Icon die einzelnen Gruppen auf. Dort finden Sie dann ein Dropdown-Menü wo Sie den gewünschten Auftrag verknüpfen können.
