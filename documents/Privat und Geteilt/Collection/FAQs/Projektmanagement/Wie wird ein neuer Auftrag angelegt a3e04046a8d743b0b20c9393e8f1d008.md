# Auftrag anlegen

---

## 1. Möglichkeit

Oben in der Menüleiste finden Sie unter dem Punkt `Projekte` einen Punkt mit `Aufträge`.  In diesem Menü wählen Sie ein Projekt aus, für welches Sie einen Auftrag erstellen wollen. Drücken Sie `+ Auftrag` um einen neuen Auftrag zu erstellen. Im Auftrag müssen Sie alle wichtigen Felder befüllt und Eigen- bzw. Fremdleistungen hinzugefügt. 

## 2. Möglichkeit

Auf der rechten Seite in der Schnellmenü-Leiste finden Sie ein großes Plus. Mit diesem Plus können Sie einige oft benötigte Funktionen aufrufen. Darunter auch ein Angebot. Wenn Sie auf das `+` und dann auf `Auftrag` drücken, öffnet sich wieder das Fenster in dem Sie alle erforderlichen Funktionen eingeben können.

## 3. Möglichkeit

Voraussetzung für diese Möglichkeit ist es, dass Sie bereits ein Angebot zu diesem Auftrag erstellt haben. Im Angebot können Sie den Status von Offen auf Beauftragt ändern. Daraus wird dann ein neuer Auftrag erstellt.
