# Wie kann ich ein Angebot aus einer bereits angelegten Vorlage erstellen?

Last Updated: 16. Oktober 2023
Published: Yes
Suggested: No

Um ein Angebot zu duplizieren, navigieren Sie zu Projekte → Angebote. Dort wählen Sie das gewünschte Projekt aus, aus in welchem Sie ein Angebot duplizieren möchten. Im nächsten Schritt müssen Sie den Haken bei „aus Vorlage“ setzen und durch + Angebot könne Sie dann die das gewünschte Angebot auswählen, das Sie duplizieren möchten.
