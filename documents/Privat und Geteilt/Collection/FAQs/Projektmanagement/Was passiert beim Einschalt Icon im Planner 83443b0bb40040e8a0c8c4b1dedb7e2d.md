# Einschalt Icon im Planner

---

Durch einen Doppelklick auf die Aufgabe öffnet sich ein Notizfeld. In diesem Notizfeld können ToDo Listen hinzugefügt werden.

![Untitled](Was%20passiert%20beim%20Einschalt%20Icon%20im%20Planner/Untitled.png)

Mit dem Einschalt Icon wird entschieden, ob die Tasks im Kanban erscheinen oder nicht. Das macht beispielsweise Sinn, wenn noch auf Infos gewartet wird.
