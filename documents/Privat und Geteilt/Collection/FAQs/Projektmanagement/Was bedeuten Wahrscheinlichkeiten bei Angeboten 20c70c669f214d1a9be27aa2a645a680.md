# Wahrscheinlichkeiten bei Angeboten

---

Wahrscheinlichkeiten haben eine Auswirkung.

Allerdings müssen Sie dies erst unter `Einstellungen → CRM → Einstellungen` aktivieren. Dafür wählen Sie die Checkbox `*„Lead Einschätzung → Auftragswert anteilig nach Wahrscheinlichkeit berechnen“*` aus.

### Beispiel:

Wenn Sie ein Angebot i.d.H.v. € 10.000,- mit einer 50 %igen Wahrscheinlichkeit haben, wird das Angebot im Forecast mit 50 % bewertet. Daher wird Ihnen der Forecast im Reporting mit € 5.000,- angezeigt.
