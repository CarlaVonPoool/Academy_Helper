# Rechnungsdokument erstellen

---

## 1. Möglichkeit

Sie drücken in der Rechnung beim Raster Dokumente auf das kleine `+`.

![Untitled](Wie%20wird%20ein%20Rechnungsdokument%20erstellt/Untitled.png)

## 2. Möglichkeit

Sie drücken im Raster Dokumente auf den Button `+ Dokumente`.

![Untitled](Wie%20wird%20ein%20Rechnungsdokument%20erstellt/Untitled%201.png)

## 3. Möglichkeit

Im Auftrag kann aus der Auftragsplanung eine Rechnung erstellt werden. Dazu navigieren Sie zur Auftragsplanung und drücken auf das + und können eine Rechnung erstellen.

![Untitled](Wie%20wird%20ein%20Rechnungsdokument%20erstellt/Untitled%202.png)
