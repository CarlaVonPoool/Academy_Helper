# Angebot anlegen

---

## 1. Möglichkeit

Oben in der Menüleiste finden Sie unter dem Punkt `Projekte` einen Punkt mit `Angebote`.  In diesem Menü können Sie ein Projekt ausgewählt, für welches Sie ein Angebot erstellen wollen. Wenn Sie auf `+ Angebot` klicken, wir dein neues Angebot erstellt.  Im Angebot können Sie alle wichtigen Felder befüllen und Eigen- bzw. Fremdleistungen hinzugefügt. 

## 2. Möglichkeit

Auf der rechten Seite in der Schnellmenü-Leiste finden Sie ein großes Plus. Mit diesem Plus können Sie einige oft benötigte Funktionen aufrufen. Darunter auch ein Angebot. Wenn Sie auf das `+` und dann auf `Angebot` drücken, öffnet sich wieder das Fenster in dem alle erforderlichen Funktionen eingegeben werden können.
