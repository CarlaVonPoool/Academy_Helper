# Ausgangsrechnung verbuchen

---

Um eine Ausgangsrechnung zu verbuchen navigieren Sie zu `Buchhaltung → Ausgangsrechnung`. Dort wechseln Sie auf den Raster `Buchen` und wählen Sie die gewünschte Rechnung aus. Jetzt können Sie die Rechnung verbuchen.
