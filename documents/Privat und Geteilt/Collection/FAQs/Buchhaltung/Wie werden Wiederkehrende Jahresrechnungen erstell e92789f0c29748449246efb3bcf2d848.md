# Wie werden Wiederkehrende Jahresrechnungen erstellt?

Published: Yes
Suggested: No

Unter **Buchhaltung - Eingangsrechnung oder Ausgangsrechnung-Wiederkehrende** findet Ihr die Einstellung zur Wiederkehrenden Rechnung.

Stelle die Frequenz auf **Monatlich, benutzerdefiniert** wähle den Zeitraum für die Rechnung. 
Bei der Durchführung tragen Sie **12 Monate** ein und entscheiden an welchem Tag in diesem Monat die Rechnung erstellt werden soll. Mit diesen Einstellungen haben Sie erfolgreich eine Wiederkehrende Jahresrechnung hinterlegt.
