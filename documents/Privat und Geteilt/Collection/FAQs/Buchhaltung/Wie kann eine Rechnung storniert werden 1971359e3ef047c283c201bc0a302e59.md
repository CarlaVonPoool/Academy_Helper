# Rechnung stornieren

---

Unter `Buchhaltung → Ausgangsrechnungen` kann die gewünschte Rechnung dupliziert werden. Dazu müssen Sie das Projekt auswählen und den Haken bei `aus Vorlage` setzen.

![Untitled](Wie%20kann%20eine%20Rechnung%20storniert%20werden/Untitled.png)

Dann müssen Sie mit `+ Ausgangsrechnung` eine neue Ausgangsrechnung erstellen und die Rechnung wählen die Sie duplizieren bzw. stornieren wollen. Im nächsten Schritt müssen Sie noch vor die Beträge ein Minus setzen. Zum Schluss können Sie wie gewohnt eine Rechnung bzw. eine Stornorechnung erstellen und versenden.
