# Wie werden Eingangsrechnungen einem Projekt/Auftrag zugewiesen?

Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

# Kosten zuweisen

---

In poool weisen Sie Kosten einer Eingangsrechnung einem Projekt oder Auftrag zu. Erstellen Sie dazu unter `Buchhaltung -> Eingangsrechnungen` eine neue Rechnung und füllen die Pflichtfelder aus. Sie können auch eine bereits bestehende Eingangsrechnung öffnen. Scrollen Sie nach unten zu den Positionen und klappen Sie mit dem Pfeil nach unten die Einstellungen zu den einzelnen Positionen auf. 

![image.png](Wie%20werden%20Eingangsrechnungen%20einem%20Projekt%20Auftra/image.png)

Wählen Sie im Feld `Projekt` das gewünschte Projekt. Wenn Sie wünschen, können Sie auch noch weitere Informationen wie die Phase, den Auftrag, das Budget oder sonstiges erfassen . Markieren Sie  die Kosten als "abrechenbar", um die Kosten später dem Kunden weiterverrechnen zu können.

![Untitled](Wie%20werden%20Eingangsrechnungen%20einem%20Projekt%20Auftra/Untitled.png)

Wenn sie dann eine Rechnung für das gewünschte Projekt erstellen, können Sie die abrechenbaren Kosten unter Fremdkosten abrechnen.
