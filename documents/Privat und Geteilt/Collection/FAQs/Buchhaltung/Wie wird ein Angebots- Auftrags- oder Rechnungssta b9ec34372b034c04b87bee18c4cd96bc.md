# Angebots-/Auftrags- oder Rechnungsstatus ändern

---

Der Status kann sowohl bei Angeboten, Aufträgen und Rechnungen in der Kalkulation geändert werden. 

![Untitled](../Projektmanagement/Wie%20wird%20ein%20Angebots-%20Auftrags-%20oder%20Rechnungssta/Untitled.png)

Wenn Sie beim Angebot den Status von Offen auf Beauftragt ändern, wird automatisch ein Auftrag erstellt.

![Untitled](../Projektmanagement/Wie%20wird%20ein%20Angebots-%20Auftrags-%20oder%20Rechnungssta/Untitled%201.png)

Sobald eine Rechnung aus einem Auftrag erstellt wird, und diese abgerechnet wird, springt auch der Status auf abgerechnet.

![Untitled](../Projektmanagement/Wie%20wird%20ein%20Angebots-%20Auftrags-%20oder%20Rechnungssta/Untitled%202.png)

Wenn eine Rechnung noch nicht abgerechnet wurde, handelt es sich noch um eine Entwurfsrechnung. Sobald diese abgerechnet wird, ändert sich der Status auf Offen. Wenn Sie die Rechnung dann buchen, ändert sich der Status auf Bezahlt. Der Status Storniert können Sie bei der Rechnung mit dem Klick auf Storniert selbst einstellen.
