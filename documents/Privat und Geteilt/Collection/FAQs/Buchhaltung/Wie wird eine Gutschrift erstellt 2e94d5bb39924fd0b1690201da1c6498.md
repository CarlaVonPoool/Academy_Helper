# Gutschrift erstellen

---

Erstellen Sie wie gewohnt unter `Buchhaltung - Ausgangsrechnung` eine ***Ausgangsrechnung.*** Wenn Sie die Ausgangsrechnung mit „***aus Vorlage***“ erstellen brauchen Sie nichts mehr ändern – Sie müssen lediglich ein ***Minus vor dem Betrag*** setzen.

Es wäre vom Vorteil, wenn Sie beim Rechnungstitel einen Vermerkt bzgl. der Gutschrift hinterlegst. 

Beispiel: ***GS- Rechnungstitel –*** somit sehen Sie auf einem Blick was eine Gutschrift und was eine Rechnung ist.

## **Hinweis:**

Gutschriften sind Negativbeträge (0 oder kleiner), weshalb Poool automatisch Gutschriften als *abgerechnet/bezahlt* hinterlegt bzw. bucht.

Die Gutschrift wird von Ihnen bezahlt und ist somit für Sie ein Negativbetrag, welcher auch beglichen werden muss.
