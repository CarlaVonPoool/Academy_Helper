# Bereits erfasste Eingangsrechnungen einer Phase zuweisen

---

Dazu gehen Sie im gewünschten Projekt zum Reiter `Kosten`. Dort finden Sie alle Kosten, die diesem Projekt zugeordnet wurden. Mit dem `Bearbeiten` Button können Sie die Kosten einer anderen Phase zuordnen.
