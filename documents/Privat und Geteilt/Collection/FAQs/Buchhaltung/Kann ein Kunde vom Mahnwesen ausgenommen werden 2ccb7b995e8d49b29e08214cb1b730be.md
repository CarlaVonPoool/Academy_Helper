# Kunde vom Mahnwesen ausnehmen

---

Suchen Sie unter `Crm → Adressbuch` den gewünschten Kontakt. Gehen Sie auf den Raster `Kunde` und drücken Sie auf `bearbeiten`. Aktivieren Sie das Kontrollkästchen bei *Vom Mahnlauf ausnehmen*.
