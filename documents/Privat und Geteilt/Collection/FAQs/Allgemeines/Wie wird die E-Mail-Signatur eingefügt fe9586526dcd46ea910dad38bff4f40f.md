# E-Mail-Signatur einfügen

---

E-Mail-Signaturen können Sie im Reiter `CRM → Adressbuch → Eure Firma / Personen` hinzufügen. Dort können Sie die Person bearbeiten und ein Feld hinzufügen (rechts oben). Die E-Mail-Signatur kann dort hineinkopiert werden.

![Untitled](Wie%20wird%20die%20E-Mail-Signatur%20eingef%C3%BCgt/Untitled.png)

Wichtig ist, dass bei den E-Mail-Vorlagen (`Einstellungen → E-Mail-Vorlagen`) der Platzhalter “{`EMAIL_SIGNATUR`}” enthalten ist. Ohne diesen Platzhalter wird die E-Mail-Signatur in Mails aus Poool nicht versendet.
