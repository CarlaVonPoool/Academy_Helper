# Liste von aktuellen Gleitzeitkonten erstellen

---

Um die aktuellen Gleitzeitkonten abzurufen, können Sie unter `Unternehmen → Mitarbeiterzeiterfassung` eine Excel-Liste exportieren. Im zweiten Raster `Export` können Sie die gewünschten Mitarbeiter und der gewünschte Zeitraum angeben. Mit dem Button `→ Arbeitszeit (xlsx)` kann eine Liste mit den Arbeitszeiten inklusive der Gleitzeit exportiert werden.
