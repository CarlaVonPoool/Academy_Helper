# Dashboard erstellen

---

Navigieren Sie zum Dashboard-Bereich in der Software und klicken Sie auf die Schaltfläche `+ Dashboard`.

![Untitled](Wie%20kann%20ein%20neues%20Dashboard%20erstellt%20werden/Untitled.png)

Auf dem Dashboard können Sie dann durch Doppelklick oder `+ Widget` neue Widgets hinzufügen und Ihr Dashboard nach Ihren Wünschen gestalten.
