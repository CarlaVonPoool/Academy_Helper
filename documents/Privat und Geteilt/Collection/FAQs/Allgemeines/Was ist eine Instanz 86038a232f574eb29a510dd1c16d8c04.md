# Was ist eine Instanz?

Published: Yes
Suggested: No

Eine Instanz in Poool ist eine separate, unabhängige Ausführungseinheit der Poool-Software. Jede Instanz ist eine isolierte Umgebung, die alle Funktionen und Datenbanken enthält, die für den Betrieb von Poool notwendig sind. Instanzen ermöglichen die Isolierung von Prozessen, die Ressourcenteilung und die Anpassung an veränderte Geschäftsanforderungen.

### Instanzen aktivieren/deaktivieren

Unter `Benutzer → Verwaltung` können die einzelnen Instanzen aktiviert und deaktiviert werden. Eine der aktiven Instanzen kann hier auch als Standard festgelegt werden.

![Untitled](Was%20ist%20eine%20Instanz/Untitled.png)

### Instanzen wechseln

Wenn für einen Mitarbeiter mehrere Instanzen aktiviert sind, können oben rechts bei dem Zahnrad die verschiedenen Instanzen eingesehen werden. Mit einem Klick auf eine andere Instanz, wechselt Poool in die gewünschte Instanz.
