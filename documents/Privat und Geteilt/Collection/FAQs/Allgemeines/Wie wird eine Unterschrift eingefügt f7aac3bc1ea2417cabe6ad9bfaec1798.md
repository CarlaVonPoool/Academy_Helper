# Unterschrift einfügen

---

Unterschriften können nur durch den Poool Support angelegt werden. Gerne können wir dies für Sie umsetzen. Bitte schicken Sie uns die jeweiligen Unterschriften (*Format: png. / jpg.*) - wir passen die Dokumentenvorlage gerne für Sie an.
