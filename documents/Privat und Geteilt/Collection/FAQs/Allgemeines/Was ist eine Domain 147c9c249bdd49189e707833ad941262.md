# Was ist eine Domain?

Published: Yes
Suggested: No

Eine **Domain** ist ein weltweit einmaliger, eindeutiger Name für einen logisch abgegrenzten Teilbereich des Internets. Sie dient dazu, Inhalte im Web zu adressieren. Zum Beispiel könnte eine Domain so aussehen: **www.alfredo-pizzeria.com**.

Die Übersetzung von Domains in IP-Adressen erfolgt durch spezialisierte Nameserver. Diese funktionieren ähnlich wie eine Telefonauskunft: Wenn du eine Domain in deinen Webbrowser eingibst, sendet der Browser eine Anfrage an den zuständigen Nameserver, der die hinterlegte IP-Adresse zurückgibt. So wird die gewünschte Ressource gefunden.
