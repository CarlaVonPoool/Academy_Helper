# Buchhaltungsexport

Article Description: Schnell und einfach den Buchhaltungsexport erstellen. 
Published: Yes
Suggested: No

Um Ihre Buchhaltung nun weiterzuleiten, müssen Sie einen Export erstellen und diesen dann versenden.

## **Export**

Den DATEV-Export finden Sie unter dem Reiter `Buchhaltung → Export`. Mit dem Button `→ DATEV Datenübermittlung` gelangen Sie zur Export-Maske. 

![Untitled](Buchhaltungsexport/Untitled.png)

Dort wählen Sie den gewünschten Zeitraum aus und können dort mit dem Button `→ Exportieren` alle Rechnungsarten als ZIP-Datei exportieren.

![Untitled](Buchhaltungsexport/Untitled%201.png)

Bereits fertige Exporte finden Sie unter dem Reiter `Exporte`. Diese können Sie im Nachhinein durch den runden Pfeil erneut als ZIP-Datei downloaden.

![Untitled](Buchhaltungsexport/Untitled%202.png)

Sie können die exportiere Datei dann Ihrem:r Steuerberater:in für die Buchung übergeben.
