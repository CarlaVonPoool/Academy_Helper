# Sepa-Zahlungsanweisung

Article Description: Alle Zahlungen auf einen Blick und nie wieder etwas vergessen.
Published: Yes
Suggested: No

Effizientes **Finanzmanagement** ist entscheidend für jedes Unternehmen. In diesem Guide erfahren Sie, wie Sie mithilfe von Poool eine **SEPA-Zahlungsanweisung** erstellen können. Dieser Prozess erleichtert nicht nur die Buchführung, sondern trägt auch dazu bei, den **Zahlungsverkehr** transparenter und zeitsparender zu gestalten. 

## Vorbereitung

Um in Poool eine SEPA-Zahlungsanweisung zu **erstellen**, gehen Sie zunächst ins `CRM` zu `Adressbuch`. Wählen Sie den **Lieferanten** aus, für den Sie in Zukunft eine SEPA-Zahlungsanweisung **erstellen** wollen und navigieren Sie im oberen Menü zu den drei Punkten wo mit einem Klick der Menüpunkt `Zahlungsmittel` erscheint. 

![Untitled](Sepa-Zahlungsanweisung/Untitled.png)

Hinterlegen Sie im Raster `Zahlungsmittel` das **Bankkonto** Ihres Lieferanten und setzen Sie den **Haken** bei “Sepa Zahlungsanweisung bei Buchung erstellen”. 

![Untitled](Sepa-Zahlungsanweisung/Untitled%201.png)

Dadurch wird die **automatische** Generierung von SEPA-Zahlungsanweisungen ermöglicht. 

## Buchung

Wenn Sie eine Rechnung **buchen** wollen, navigieren Sie zu `Buchhaltung → Eingangsrechnung` und wechseln Sie zu dem Raster `Buchen`. Öffnen Sie die **Rechnung**, die Sie buchen wollen. Setzen Sie den Haken bei **SEPA-Zahlungsanweisung** und wählen Sie einen **Empfänger** der Zahlung aus. Wenn Sie das **Bankkonto** nicht im Vorhinein im **CRM** hinterlegt haben, können Sie auch **ohne Empfänger** weitermachen. Geben Sie einen **IBAN** und einen **BIC** ein und wenn gewünscht auch eine **Zahlungsreferenz**. Wenn Sie keine Zahlungsreferenz eingeben, wird immer automatisch die **Rechnungsnummer** hergezogen. Sie können in diesem Schritt auch den **Betrag** anpassen, den Sie überweisen möchten. Sobald Sie alle **gewünschten Daten** eingegeben haben, können Sie die Rechnung buchen. 

![Untitled](Sepa-Zahlungsanweisung/Untitled%202.png)

## Export

Nachdem Sie die Rechnung **gebucht** haben, wechseln Sie zu `Buchhaltung → Export`. Öffnen Sie `→ SEPA Zahlungsanweisungen` und wählen Sie einen **Zeitraum** für den Sie die Zahlungsanweisung **exportieren** wollen. 

![Untitled](Sepa-Zahlungsanweisung/Untitled%203.png)

Sollte der IBAN, der BIC oder die Kombination **falsch** eingetragen sein, wird das in diesem Schritt **angezeigt**. Drücken Sie auf die Zeile mit dem falschen **Eintrag** und passen Sie Ihn an. Wenn alle Daten **stimmen**, können Sie mit dem Button `→ Exportieren (xml)` die Datei **exportieren**. 

![Untitled](Sepa-Zahlungsanweisung/Untitled%204.png)

## Erneut exportieren

Wenn Sie eine bereits **exportierte Liste** erneut exportieren wollen, navigieren Sie zu `Buchhaltung → Export` und wählen sie `→ SEPA Zahlungsanweisungen`. Wechseln Sie zum Raster `Exporte`, um Ihre bereits exportierten **Dateien** zu sehen. 

![Untitled](Sepa-Zahlungsanweisung/Untitled%205.png)

Mit einem Klick auf die **Download-Spalte** des Exports können Sie die bereits exportierte Datei **erneut** exportieren. 

Mit diesen einfachen Schritten **optimieren** Sie nicht nur Ihre finanziellen Abläufe, sondern gewährleisten auch die **Genauigkeit** und **Relevanz** Ihrer SEPA-Zahlungsanweisung. Nutzen Sie die Funktionen, um Ihren Zahlungsverkehr zu **vereinfachen** und Zeit für das Wesentliche zu **gewinnen**.
