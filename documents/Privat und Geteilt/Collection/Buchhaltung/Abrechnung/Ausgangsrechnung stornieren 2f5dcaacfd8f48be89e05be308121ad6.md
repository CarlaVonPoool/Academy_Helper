# Ausgangsrechnung stornieren

Article Description: Automatische Erstellung einer Stornorechnung zu einer bestehende Ausgangsrechnung.
Last Updated: 9. Oktober 2023
Published: Yes
Suggested: No

Wenn eine Ausgangsrechnung storniert werden muss, wird im gleichen Zuge auch eine Stornorechnung bzw. eine Gutschrift benötigt. 

## Rechnung stornieren

Suchen Sie sich die Rechnung heraus, welche Sie stornieren möchten und öffnen Sie die Rechnung. Sie finden **neben** dem **Rechnungsdatum** einen Button `→ Stornieren`. Klicken Sie auf den Button, um die Rechnung zu stornieren.

![Untitled](Ausgangsrechnung%20stornieren/Untitled.png)

Wenn Sie auf den Button drücken, werden Sie gefragt, ob Sie eine Stornorechnung erstellen möchten. Sie können im **selben Schritt** auch entscheiden, ob Sie die **Ausgangsrechnung** und die **Stornorechnung verbuchen** wollen. Wenn Sie in diesem Dialogfeld auf `→ OK` drücken, wird die Rechnung storniert und gleichzeitig wird eine Stornorechnung erstellt und die gewünschte **Ausgangsrechnung bekommt** den Status `Storniert`. Je nachdem wie Sie die Haken gesetzt haben, werden die Rechnungen dann auch verbucht. 

![Untitled](Ausgangsrechnung%20stornieren/Untitled%201.png)

## Stornorechnung bearbeiten

Im nächsten Schritt **öffnet** sich die **Stornorechnung als Entwurf** mit dem gesamten negativen Rechnungsbetrag. Sie können die Stornorechnung dann Ihren **Wünschen entsprechend anpassen**. Die Stornorechnung bekommt den **Titel der stornierten Ausgangsrechnung** mit dem Zusatz **STORNO**. Sie können diesen Zusatz auch ändern, beispielsweise in Gutschrift. 

## Stornorechnung erstellen

Wenn Sie mit der Kalkulation der Stornorechnung bzw. der Gutschrift zufrieden sind, können Sie die Rechnung abrechnen. Dazu drücken Sie auf den Button `→ Abrechnen`, um der Rechnung eine Nummer zu vergeben.

![Untitled](Ausgangsrechnung%20stornieren/Untitled%202.png)

Im nächsten Schritt können Sie wie gewohnt ein Dokument für die Stornorechnung/Gutschrift erstellen und versenden.
