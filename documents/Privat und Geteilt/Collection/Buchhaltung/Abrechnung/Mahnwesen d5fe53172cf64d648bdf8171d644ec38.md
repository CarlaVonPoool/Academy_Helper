# Kunden Mahnen

Navigieren Sie zu `Buchhaltung → Mahnwesen`.

![Untitled](Mahnwesen/Untitled.png)

Im Raster neue Fälligkeiten sehen Sie alle Rechnungen die bis zum heutigen Datum fällig gewesen wären und noch kein Zahlungseingang war. Das bedeutet Sie haben für diese Rechnungen in Poool noch keinen Zahlungseingang verbucht. Sie sehen auch, wann die Rechnung fällig gewesen wäre und auf welcher Mahnstufe sich Ihr Kunde befindet. 

![Untitled](Mahnwesen/Untitled%201.png)

### Mahnlauf starten

Um den Mahnlauf zu starten müssen Sie den Button `+ Mahnlauf` drücken. Im nächsten Schritt erscheint ein Kontrollfeld indem Sie den Titel des Mahnlaufs anpassen können.

![Untitled](Mahnwesen/Untitled%202.png)

Der Mahnlauf wird erstellt und Sie können wählen, welche Mahnungen sie versenden wollen und welche nicht. Das entscheiden Sie über das Symbol auf der rechten Seite bei “Aktion”. 

![Untitled](Mahnwesen/Untitled%203.png)

Standartmäßig ist bei allen Rechnungen das Rechnungs-Symbol hinterlegt. Das bedeutet, dass ein Rechnungsdokument erstellt wird und die Mahnstufe erhöht wird.

![Untitled](Mahnwesen/Untitled%204.png)

Klicken Sie einmal auf das Symbol erscheint ein +. Das bedeutet, dass die Mahnstufe erhöht wird aber kein Dokument erstellt wird.

![Untitled](Mahnwesen/Untitled%205.png)

Klicken Sie ein weiteres Mal auf das Symbol erscheint ein durchgestrichener Kreis. Das bedeutet, dass die Rechnung in diesem Mahnlauf ignoriert wird und wieder unter den neuen Fälligkeiten angezeigt wird.

![Untitled](Mahnwesen/Untitled%206.png)

Wenn Sie bei allen Rechnungen das gewünschte Symbol ausgewählt haben, können Sie mit dem Button `→ Starten` den Mahnlauf starten. Die gewünschten Aktionen werden dann ausgeführt. Mit einem Klick auf die Rechnung können dann noch die Mahngebühren hinzugefügt oder angepasst werden.

![Untitled](Mahnwesen/Untitled%207.png)

Im nächsten Schritt können Sie die Mahnungen mit `→ Mahnungen (zip)` als PDFs herunterladen und versenden.

![Untitled](Mahnwesen/Untitled%208.png)
