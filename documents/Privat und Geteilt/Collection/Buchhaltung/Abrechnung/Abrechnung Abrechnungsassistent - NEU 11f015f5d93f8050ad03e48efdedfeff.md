## Abrechnungsassistent

Der **Abrechnungsassistent** (Zauberstab-Symbol) in Poool bietet eine effiziente Möglichkeit zur **Abrechnung** von Aufträgen, Fremdkosten und Stunden. Diese **Anleitung** führt Sie durch die verschiedenen Funktionen und Schritte, um Rechnungen effektiv zu erstellen und zu verwalten.

## Vorbereitung

### Auftragsplanung

Im gewünschten Projekt können Sie unter dem Tab **Auftragsplanung** einen **Abrechnungsplan** anlegen. Dabei entscheiden sie, ob nach **Gruppen** oder **Positionen** abgerechnet werden soll. Sie können außerdem Bezeichnungen und Fälligkeiten individuell anpassen. Sobald sie speichern, wird der Abrechnungsplan aktiv und kann im Assistenten verwendet werden. 

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image.png)

### Abrechnung von Fremdkosten

Fremdkosten werden über **Eingangsrechnungen** im System erfasst. Einzelne Positionen können sie dabei einem Projekt zuordnen und als **abrechenbar** markieren. Nur diese erscheinen später im Abrechnungsassistenten.

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%201.png)

### Abrechnung von Stunden und Tickets

Grundlage für das Abrechnen von Stunden ist die **Zeiterfassung** der Mitarbeiter und die Einstellung der **Projekte**. 

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%202.png)

Informationen zur Bewertung von Stunden und die entsprechenden **Einstellungen** finden Sie im **Stundenmanagement**.

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%203.png)

### Abrechnen mit dem Abrechnungsassistenten

Sobald die Vorbereitung abgeschlossen ist, öffnen Sie eine bestehende Rechnung oder legen eine neue unter `Buchhaltung → Ausgangsrechnung` an. Über das **Zauberstab-Symbol** gelangen Sie in den Abrechnungsassistenten.

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%204.png)

Wählen Sie das gewünschte **Projekt** aus (standardmäßig wird das aktuelle vorgeschlagen) und **entscheiden** Sie sich für eine der drei Abrechnungsarten: 

- mit Hilfe der Auftragsplanung
- mit Hilfe der Fremdkosten
- mit Hilfe der Stunden und Tickets

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%205.png)

Die jeweils verfügbaren **Positionen** werden Ihnen anschließend zur Auswahl angezeigt.

Sie können die Positionen einzeln auswählen, bearbeiten oder kombinieren. Dabei lassen sich Beträge, Prozentsätze und Verrechnungen flexibel anpassen – sowohl bei Eigen- als auch bei Fremdleistungen.

- **Auftragsplanungen zusammengefasst** (alle Positionen aus der Auftragsplanung werden zu einer Position zusammengefasst)

### Erweitertes Hinzufügen

Wenn Sie **Positionen** gezielt einer bestehenden Gruppe oder Rechnungsposition zuordnen möchten, nutzen Sie die Funktion ****`Erweitertes Hinzufügen`. Im folgenden **Dialog** können Sie neue Gruppen anlegen, bestehende **erweitern** oder Beträge zu vorhandenen Positionen hinzufügen

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%2011.png)

Dort können Sie mit dem kleinen `+` auf der Seite wählen, ob Sie für die gewählte/n **Position/en** eine neue **Gruppe** erstellen wollen, eine Position in einer **bereits bestehenden** Gruppe erstellen wollen oder den **Betrag** zu einer bereits bestehenden Position **dazurechnen** wollen. Mit dem Button `Zur Rechnung hinzufügen`, fügen Sie dann die **Position** wie gewünscht zur Rechnung hinzu.

![image.png](Abrechnung%20Abrechnungsassistent%20-%20NEU/image%2012.png)

Der **Abrechnungsassistent** bietet Ihnen eine einheitliche Oberfläche, um Auftragsleistungen, Fremdkosten oder Stunden einfach und **strukturiert abzurechnen**. Sobald die entsprechenden Grundlagen im System erfasst sind, können Sie die **relevanten Positionen** mit wenigen Klicks zur Rechnung hinzufügen und flexibel anpassen.
