# Auftragsplanung

Wenn Sie in einen gewünschten Auftrag auf die **Auftragsplanung** klicken, können Sie einen **Abrechnungsplan** auswählen. Sie können zwischen "**nach Gruppe**" und "**nach Positionen**" wählen. Wie der Name bereits sagt, können Sie hier nach Positionen oder Gruppen abrechnen. Dort können Sie die Bezeichnungen und Fälligkeiten gegebenenfalls anpassen. Sobald Sie gespeichert haben, wird der Abrechnungsplan aktiv.

# Rechnungserstellung

Im nächsten Schritt klicken Sie auf **Rechnungen**, erstellen durch + Rechnung eine Rechnung ****und rechnen über den **Abrechnungsassistenten** (auch bekannt als Zauberstab) die Auftragsplanung ab.

Dort können Sie auswählen, ob Sie eine einzelne Position abrechnen wollen oder beispielsweise zwei Positionen abrechnen möchten. 

Im nachfolgenden Beispiel werden Beratung (EUR 1.760,-), Programmierung (EUR 1.920,-) sowie Korrekturen (EUR, 240,-) abgerechnet.

![Untitled](Abrechnen%20nach%20Position/Untitled%201.png)

Es ist wichtig, dass Sie nun auf "**Auftragsplanungen einzeln"** klicken und im nächsten Schritt "zur Rechnung hinzufügen" auswählen.

Dadurch werden die beiden Positionen in der Rechnung als Gruppe zusammengefasst dargestellt. Die Gruppen- oder Positionsbeschreibungen werden 1:1 vom Auftrag übernommen.
