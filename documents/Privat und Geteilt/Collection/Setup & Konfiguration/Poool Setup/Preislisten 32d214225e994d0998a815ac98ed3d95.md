# Preislisten

Last Updated: 18. August 2025
Published: Yes

Wenn Sie eine neue **Preisliste** erstellen möchten oder Änderungen an einer bestehenden Preisliste vornehmen möchten, können Sie dies ganz einfach unter `Einstellungen → Preise & Funktionen` tun. Hier finden Sie alle notwendigen Optionen, um eine Preisliste zu erstellen, einschließlich der Möglichkeit, neue **Funktionsgruppen** und **Funktionen** hinzuzufügen, die Preise für jede Funktion festzulegen und die Preisliste an Ihre Bedürfnisse anzupassen. 

![preise-1.png](Preislisten/preise-1.png)

## **Funktionsgruppen**

Um eine neue Funktionsgruppe zu erstellen, klicken Sie auf den Reiter `Funktionsgruppen` und wählen Sie die Option `+ Funktionsgruppe`. Auf diese Weise können Sie eine **Gruppe** erstellen, die später in der Preisliste unter den jeweiligen **Funktionsgruppen** angezeigt wird. Dort finden Sie dann alle einzelnen Funktionen, die innerhalb der Gruppe verfügbar sind. Es ist wichtig, die Funktionen sorgfältig auszuwählen und ihre **Einzigartigkeit** hervorzuheben, um sicherzustellen, dass sie für potenzielle Kunden attraktiv sind. 

![preise-2.png](Preislisten/preise-2.png)

## Funktionen

Um eine neue **Funktion** in der Preisliste zu erstellen, klicken Sie auf die Registerkarte `Funktionen`. Dort können Sie durch die Schaltfläche `+ Funktion` eine neue Funktion erstellen. Es ist jedoch wichtig, dass Sie die neu erstellte Funktion mithilfe des Kontrollkästchens `Aktiv` aktivieren. Nur so wird die Funktion in der Preisliste angezeigt und kann von Kund:innen ausgewählt werden. Vergessen Sie nicht, dass die Funktionen in der Preisliste ein wichtiger Bestandteil Ihres Angebots sind und eine klare und detaillierte **Beschreibung** erfordern, damit Kunden genau verstehen, was sie erhalten und zu welchem **Preis**. 

![preise-3.png](Preislisten/preise-3.png)

Eine weitere wichtige Information bei der Erstellung von Preislisten ist, dass Sie die **Tagespreise** stets durch die **Anzahl der Stunden** pro Tag dividieren sollten. Dies stellt sicher, dass die Preise im Einklang mit den tatsächlich geleisteten **Arbeitsstunden** stehen und somit eine faire und transparente **Abrechnung** gewährleistet wird. Indem Sie diese Berechnungsmethode anwenden, können Sie sicherstellen, dass Ihre Kunden eine klare Vorstellung davon haben, wie viel sie für Ihre Dienstleistungen bezahlen müssen.

![preise-4.png](Preislisten/preise-4.png)

## Preisliste

Durch die Verwendung der Funktion `Preisliste` in unserem System haben Sie die Möglichkeit, schnell und einfach eine neue **Preisliste** zu erstellen. Sie können dabei den **Titel** der Preisliste und ein **Gültigkeitsdatum** angeben. Darüber hinaus können Sie auch weitere Informationen wie zum Beispiel **Preisänderungen, Rabatte** oder **neue Produkte** einfügen. 

![preise-5.png](Preislisten/preise-5.png)

Es gibt verschiedene **Funktionen**, die Sie mit der Checkbox auf der linken Seite entfernen oder hinzufügen können. Wenn Sie z.B. eine Funktion **entfernen**, können Sie sicherstellen, dass keine **unerwünschten Funktionen** erscheinen. Auf der anderen Seite können Sie auch neue Funktionen hinzufügen, um das **Benutzererlebnis** zu **verbessern**. 

![preise-6.png](Preislisten/preise-6.png)
