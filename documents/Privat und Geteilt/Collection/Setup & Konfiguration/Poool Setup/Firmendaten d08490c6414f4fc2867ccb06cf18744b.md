# Eigene Firma einrichten

## Basiseinstellungen

Im Reiter `CRM → Adressbuch → bearbeiten` haben Sie die Möglichkeit, Ihren eigenen Kontakt in der Übersicht zu pflegen. Dort können Sie neben den grundlegenden Informationen wie Name, Adresse und Telefonnummer auch weitere Details zu Ihrem Kontakt hinterlegen, wie beispielsweise:

- E-Mail-Adresse
- Geburtsdatum
- Beruf
- Interessen
- Social-Media-Profile

Falls Ihnen noch Felder fehlen, können Sie diese jederzeit über die Funktion **„Feld hinzufügen“** ergänzen. Eine umfassende Pflege Ihrer Kontaktdaten kann Ihnen dabei helfen, den **Überblick über Ihre Geschäftskontakte** zu behalten und eine effektive Kundenbeziehung aufzubauen.

![Kontaktübersicht.png](Firmendaten/Kontaktubersicht.png)

**Kontakt:**

- Firmenname (intern)
- Firmenname (offiziell)
- Kürzel
- UID
- Firmenbuchnummer
- Gerichtsstand
- Geschäftsführung

**Kontaktmöglichkeiten:**

- Telefon
- E-Mail
- Webseite
- Fax

**Adressen:** 

- Adressbezeichnung
- Anschrift Zeile 1-3
- Straße / Hausnummer
- Postleitzahl / Ort
- Bundesland
- Land

![Kontaktübersicht-2.png](Firmendaten/Kontaktubersicht-2.png)

### Noch mehr Informationen

Die Kontaktmaske bietet Ihnen eine Auswahl an gängigen Begriffen um die Daten der Firma entsprechend abzubilden. Sollten Sie daran interessiert sein, Ihr Unternehmen über den Standard hinaus abzubilden, können Sie der Ansicht zusätzliche Felder hinzufügen.

Alle Felder, welche Sie dem Kontaktdatensatz anhängen, können auf den Dokumenten referenziert werden, um Ihre Rechnungen, Angebote und Auftragsbestätigungen noch individueller zu gestalten.

### Sonderfall: Geburtstage

Über die Kalenderfunktion in Poool bieten wir Ihnen einen Geburtstagskalender an, welcher auf die Daten aus dem Adressbuch zugreift.

### Das Kürzel

In Poool wird **jedem Kunden ein eigenes Kürzel** zugewiesen. Dabei handelt es sich um eine kurze Zeichenfolge, welche den **Kunden identifiziert**. Später wird dieses Kürzel als Projekt **Identifikator** dienen, damit Sie anhand der Projektnummer immer den zugeordneten Kunden erkennen können.

Dieses Kürzel tritt zum ersten Mal hier in der eigenen Firma auf, wird aber auch bei jedem neuen Kunden erneut verwendet.

## Kontaktmöglichkeiten

Hier können Sie alle relevanten Möglichkeiten zur Kontaktaufnahme sowohl für das Unternehmen, als auch für Personen im Unternehmen festhalten und über das Adressbuch leicht darauf zugreifen.

![Kontaktübersicht-3.png](Firmendaten/Kontaktubersicht-3.png)

### Adressen, Hauptanschrift

Tragen Sie bitte gewissenhaft ihre **Hauptanschrift** in die dafür vorgesehen Felder ein und vergessen Sie dabei nicht das Feld **“Anschrift Zeile 1”** damit auch der Firmenname später im Adressblock Ihrer Dokumente aufscheint.

Sollte eine abweichende Rechnungsadresse vorliegen, erstellen Sie einfach eine neue Adresse und tragen dort die entsprechenden Daten ein.

![Kontaktübersicht-4.png](Firmendaten/Kontaktubersicht-4.png)
