# Dashboard einrichten

Um ein neues **Dashboard** einzurichten, navigieren Sie im oberen rechten Eck zu den **vier Quadraten** und wählen Sie `+ Dashboard`, um ein neues Dashboard hinzuzufügen. Sie können beliebig viele Dashboards anlegen, wenn Sie beispielsweise themenspezifische Dashboards präferieren. 

[https://vimeo.com/806348166](https://vimeo.com/806348166)

Um **unterschiedliche Widgets** hinzuzufügen, drücken Sie entweder oben Rechts auf `+ Widget` oder klicken Sie doppelt auf Ihr Dashboard. So können Sie Ihr Dashboard nach Ihren Wünschen gestalten mit den einzelnen Inhalten, die für Ihre Arbeit relevant sind.

![dashboard.png](Dashboard/dashboard.png)

### Widget löschen

Um ein **Widget** vom Dashboard wieder zu **entfernen**, klicken Sie bei dem gewünschten Widget auf den **Pfeil** nach unten. In diesen **Einstellungen** können Sie den Namen des Leads ändern oder das Widget **löschen**. Je nach **Art** des Widgets, können hier noch weitere **Einstellungen** vorgenommen werden.

![image.png](Dashboard/image.png)

### Dashboard-Vorlagen

Sie haben auch die Möglichkeit, für Ihr Team oder Ihre Kollegen ein Dashboard als Vorlage anzulegen. Dabei können Sie Widgets hinzufügen, ein Hintergrundbild festlegen und das Dashboard individuell konfigurieren.

![image.png](Dashboard/image%201.png)

Über das **Viereck-Symbol** oben rechts gelangen Sie zum Zahnrad-Menü, über das sich eine Dashboard-Vorlage einrichten lässt. Dazu erstellen Sie zunächst ein Dashboard und speichern es anschließend über das Zahnrad als Vorlage ab. Setzen Sie den Haken bei **„Öffentlich“**, wird die Vorlage auch für andere nutzbar und auswählbar, wenn ein neues Dashboard angelegt wird.
