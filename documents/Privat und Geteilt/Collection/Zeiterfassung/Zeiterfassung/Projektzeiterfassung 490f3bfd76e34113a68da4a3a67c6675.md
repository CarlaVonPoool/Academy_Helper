# Projektzeiterfassung

In der Zeiterfassung können neben der regulären Arbeitszeit auch **Zeiten auf Projekte, Phasen und Tickets** erfasst werden. Sie haben dabei die Möglichkeit, Zeitaufwände für **Kundenprojekte** (Projekterfassung) oder für **interne Tätigkeiten** zu erfassen.

![image.png](Projektzeiterfassung/image.png)

In Poool gibt es **zwei Wege**, um Projektzeiten zu erfassen - im **Zeiterfassungsmodul** oder über die **Schnellzugriffleiste** am rechten Bildschirmrand mithilfe der **Stoppuhr**. 

## **📅 Option 1: Über die Zeiterfassung**

Navigieren Sie zum Reiter `Zeiterfassung` und wählen Sie nun die Projektzeit aus. Um Stunden auf ein Projekt zu erfassen, klicken Sie auf das **“+”** neben der Projekterfassung und geben Sie in der Suchleiste das Projekt ein. Das Kästchen in der Projektzeile zeigt Ihnen an, wie viele Tickets an einem Projekt hängen bzw. ob überhaupt ein Ticket vorhanden ist. Hier können Sie also direkt nach **Projekte, Phasen oder Tickets** suchen.

![Untitled](Projektzeiterfassung/Untitled.png)

Über einen Klick werden die Tickets aufgeklappt und Sie können das zu bebuchende Ticket anklicken. Das Tickets wird nun in einer neuen Zeile angeführt und es lassen sich für den jeweiligen Tag die Zeiten erfassen, welche dem Ticket zugewiesen werden sollen.

![Untitled](Projektzeiterfassung/Untitled%201.png)

In dem Feld `Kommentar Kunde` können Sie eine Aufgabenbeschreibung hinzufügen, die Sie am Ende auch auf dem Leistungsnachweis oder dem Stundenreport anzeigen lassen können. 

In Poool können Sie in einer Zeile Zeiten **für mehrere Tage** auf ein und dasselbe Projekt oder Ticket buchen. Diese Buchung wird jedoch als ein **zusammenhängendes Event** behandelt, wodurch nur ein gemeinsames Kommentarfeld für den Kunden zur Verfügung steht. Diese Vorgehensweise eignet sich vor allem dann, wenn Sie über **mehrere Tage an derselben Aufgabe** arbeiten.

Wenn Sie hingegen unterschiedliche Tätigkeiten erfassen möchten, wählen Sie das Projekt einfach erneut aus – es wird eine neue Zeile geöffnet, die als separate Zeitbuchung gilt.

![image.png](Projektzeiterfassung/image%201.png)

<aside>
⚠️

**Wichtig:** Wenn Sie Projektzeiten in einer Zeile über mehrere Tage erfassen, wird dies als ein einzelnes Event – also eine **zusammenhängende Zeitbuchung** – behandelt. Sollte es später notwendig sein, Ihre Stunden zu verschieben, beispielsweise weil sie versehentlich auf das falsche Projekt gebucht wurden, kann nur das gesamte Event verschoben werden – nicht einzelne Stunden daraus.

</aside>

### ℹ️ **Symbole in einer Zeitbuchungszeile**

In einer Eventzeile können zudem weitere Einstellungen vorgenommen werden. 

![image.png](Projektzeiterfassung/image%202.png)

| Symbol | Bedeutung |
| --- | --- |
|  | Alternative Auswahl einer Funktion für die Zeitbuchung, z.B. zur Verwendung eines anderen Stundensatzes |
| I | Internes Kommentarfeld zur internen Dokumentation |
| T | Wenn die Zeitbuchung auf **Ticketebene** erfolgt, wird dies in der Eventzeile besonders gekennzeichnet. |
| Ü | Überstunden werden bei der Zeiterfassung speziell gekennzeichnet, sodass im Überstunden- Zeitkonto sichtbar ist, welche Stunden zusätzlich zur regulären Arbeitszeit erfasst wurden. |

Eine Buchungszeile kann gelöscht werden, indem Sie auf das **Löschen-Symbol** am rechten Rand der Zeile klicken.

![image.png](Projektzeiterfassung/image%203.png)

## ⏱️ Option 2: Zeitbuchungen über die Stoppuhr

Alternative können Projektzeiten auch über die **Schnellzugriffsleiste** direkt gebucht werden.

![Untitled](Projektzeiterfassung/Untitled%202.png)

![Untitled](Projektzeiterfassung/Untitled%203.png)

![Untitled](Projektzeiterfassung/Untitled%204.png)

Klicken Sie auf die **Stoppuhr in der Schnellleiste** am rechten Rand, um direkt in eine vereinfachte Version der Zeiterfassung zu gelangen. Dort können Sie das zu buchende **Projekt** oder **Ticket** auswählen, die **Zeit** sowie das **Datum** festlegen, einen **Kundenkommentar** hinzufügen und die passende **Funktion** auswählen.

Alternativ können Sie durch einen Klick auf die Stoppuhr die Zeitbuchung **sofort starten**, ohne vorher ein Projekt auswählen zu müssen.

![image.png](Projektzeiterfassung/image%204.png)

<aside>
💡

Diese Vorgehensweise empfiehlt sich, wenn Sie beispielsweise einen Kundenanruf erhalten und die Zeit **sofort erfassen** möchten, ohne viel Zeit mit Eingaben zu verlieren. Die Auswahl des **Projekts** oder **Tickets** kann anschließend vorgenommen werden.

</aside>

# Übersichtsreiter - “Mein Tag”

## 🔍 **Was zeigt dir „Mein Tag“?**

Durch eine Progress Bar sehen Sie ihre **Tagesübersicht**, aufgeteilt in:

- **Arbeitszeit**: Die gesamte Soll-Arbeitszeit des heutigen Tags.
- **Projektzeit**: Die Zeit, die Sie bereits auf Projekte gebucht haben.
- **Buchungen**: Alle einzelnen Zeitbuchungen des aktuellen Tages – inklusive Projektnamen, Uhrzeiten und Beschreibung.

![image.png](Projektzeiterfassung/image%205.png)

So erkennen Sie sofort, wie Ihr Arbeitstag aktuell aussieht und wie viel Zeit schon erfasst wurde.

### ⏱️ **Zeiterfassung per Stoppuhr – ohne Umwege**

Auch in dieser Ansicht haben Sie die integrierte **Stoppuhr-Funktion.** Sie müssen nicht den Tag auswählen oder durch die Kalenderansicht navigieren – Sie sind ja schon auf dem heutigen Tag!

- Starten Sie die Zeiterfassung mit einem Klick – ganz ohne manuelle Auswahl eines Tages.
- Wählen Sie direkt das Projekt oder die Aufgabe aus.
- Die laufende Zeit wird automatisch mitgestoppt.

## ❌ Abgelehnte Stunden

Im Reiter abgelehnte Stunden finden Sie alle Zeitbuchungen, die beispielsweise **eine Anpassung oder Korrektur erfordern**. Je nach Ihrem definierten Workflow können bestimmte Ablehnungsgründe hinterlegt sein – zum Beispiel, wenn eine Aufgabenbeschreibung als Kundenkommentar fehlt. 

✅ Nach der Korrektur werden diese Stunden wieder im Stundenmanagement angezeigt und stehen der zuständigen Person zur erneuten Prüfung bereit.
